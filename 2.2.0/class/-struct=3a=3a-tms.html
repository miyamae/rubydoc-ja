class Struct::Tms

クラスの継承リスト: Struct::Tms < Struct < Enumerable < Object < Kernel < BasicObject

要約

Process.#times の返り値を表現する構造体です。

この機能がサポートされているプラットフォーム上でプロセスの 処理時間に関する情報を保持します。プラットフォームによっては 使えない値があります。

@see Process.#times
