class IRB::ReadlineInputMethod

クラスの継承リスト: IRB::ReadlineInputMethod < Readline < IRB::InputMethod < Object < Kernel < BasicObject

要約

readline を用いた標準入力からの入力を表すクラスです。ライブラリ内部で使 用します。readline の require に失敗した場合は定義されません。
