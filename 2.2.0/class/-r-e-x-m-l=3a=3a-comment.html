class REXML::Comment

クラスの継承リスト: REXML::Comment < Comparable < REXML::Child < REXML::Node < Object < Kernel < BasicObject

要約

XML コメントを表すクラス。

コメントとは <!-- と --> で挟まれたテキストです。 

require 'rexml/document'
doc = REXML::Document.new(<<EOS)
<!-- xx -->
<root>
<!-- yy -->
text
<!-- zz -->
</root>
EOS

doc[0].string # => " xx "
doc.root[1].string # => " yy "
doc.root[3].string # => " zz "
