class ScriptError

クラスの継承リスト: ScriptError < Exception < Object < Kernel < BasicObject

要約

スクリプトのエラーを表す例外クラスです。

以下の例外クラスのスーパークラスです。

  • LoadError
  • NotImplementedError
  • SyntaxError

これらの例外が発生したときは Ruby スクリプト自体に バグがある可能性が高いと考えられます。
