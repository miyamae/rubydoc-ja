class Net::IMAP::BodyTypeMessage

クラスの継承リスト: Net::IMAP::BodyTypeMessage < Struct < Enumerable < Object < Kernel < BasicObject

要約

Content-Type が "message" であるメッセージを表すクラスです。

メールをメールに添付した場合などに使われます。 詳しくは [RFC2045], [RFC822] を参照してください。
