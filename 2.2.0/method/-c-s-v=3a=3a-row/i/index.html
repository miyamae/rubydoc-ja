instance method CSV::Row#index

index(header, minimum_index = 0) -> Fixnum[permalink][rdoc]

与えられたヘッダの名前に対応するインデックスを返します。

[PARAM] header:
ヘッダの名前を指定します。
[PARAM] minimum_index:
このインデックスより後で、ヘッダの名前を探します。 重複しているヘッダがある場合に便利です。
