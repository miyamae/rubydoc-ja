instance method Array#each_index

each_index {|index| .... } -> self[permalink][rdoc]
each_index -> Enumerator

各要素のインデックスに対してブロックを評価します。

以下と同じです。 

(0 ... ary.size).each {|index| ....  }

ブロックが与えられなかった場合は、自身と each_index から生成した Enumerator オブジェクトを返します。
