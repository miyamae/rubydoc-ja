instance method FalseClass#&

self & other -> false[permalink][rdoc]

常に false を返します。

[PARAM] other:
論理積を行なう式です。

& は再定義可能な演算子に分類されていますので、通常は false & other の形で使われます。 

p false & true     #=> false
p false & false    #=> false
p false & nil      #=> false
p false & (1 == 1) #=> false
p false & (1 + 1)  #=> false

p false.&(true)    #=> false
p false.&(false)   #=> false
p false.&(nil)     #=> false
p false.&(1 == 1)  #=> false
p false.&(1 + 1)   #=> false
