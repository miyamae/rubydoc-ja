instance method Benchmark::Tms#-

self - x -> Benchmark::Tms[permalink][rdoc]

self と x の減算を計算します。

[PARAM] x:
Benchmark::Tms のオブジェクトか Float に暗黙の変換ができるオブジェクトです。
[RETURN]
計算結果は新しい Benchmark::Tms オブジェクトです。
