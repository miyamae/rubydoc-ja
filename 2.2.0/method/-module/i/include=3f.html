instance method Module#include?

include?(mod) -> bool[permalink][rdoc]

self かその親クラス / 親モジュールがモジュール mod を インクルードしていれば true を返します。

[PARAM] mod:
Module を指定します。 
module M
end
class C1
  include M
end
class C2 < C1
end

p C1.include?(M)   # => true
p C2.include?(M)   # => true
