instance method OpenSSL::X509::Store#add_path

add_path(path) -> self[permalink][rdoc]

path が指し示すディレクトリに含まれる証明書ファイルを 信頼する証明書として加えます。

[PARAM] path:
証明書が含まれるディレクトリのパス
[EXCEPTION] OpenSSL::X509::StoreError:
追加に失敗した場合に発生します
