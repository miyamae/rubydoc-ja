instance method IO#close_on_exec?

close_on_exec? -> bool[permalink][rdoc]

自身に close-on-exec フラグが設定されていた場合 true を返します。 そうでない場合に false を返します。 

f = open("/dev/null")
f.close_on_exec?                 #=> false
f.close_on_exec = true
f.close_on_exec?                 #=> true
f.close_on_exec = false
f.close_on_exec?                 #=> false
