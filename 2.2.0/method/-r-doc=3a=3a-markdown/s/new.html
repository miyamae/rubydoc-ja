singleton method RDoc::Markdown.new

new(extensions = DEFAULT_EXTENSIONS, debug = false)[permalink][rdoc]

引数 extensions で与えられた拡張を有効にした RDoc::Markdown オブ ジェクトを初期化します。

[PARAM] extensions:
有効にする拡張を Symbol の配列で指定します。
[PARAM] debug:
デバッグを有効にするかどうかを指定します。
