instance method Hash#has_value?

has_value?(value) -> bool[permalink][rdoc]
value?(value) -> bool

ハッシュが value を値として持つ時真を返します。 値の一致判定は == で行われます。

[PARAM] value:
探索する値を指定します。 
p({1 => "one"}.has_value?("one")) #=> true
p({1 => "one"}.has_value?("two")) #=> false
