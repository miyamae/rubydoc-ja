instance method Matrix#adjugate

adjugate -> Matrix[permalink][rdoc]

余因子行列を返します。 

require 'matrix'
Matrix[[7,6],[3,9]].adjugate # => Matrix[[9, -6], [-3, 7]]
[EXCEPTION] ExceptionForMatrix::ErrDimensionMismatch:
行列が正方でない場合に発生します。
