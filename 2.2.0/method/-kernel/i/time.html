instance method Kernel#time

time(msg, width = 25) { ... } -> object[permalink][rdoc] [added by rubygems/timer]

与えられたブロックの実行時間を計測して表示します。

[PARAM] msg:
表示するメッセージを指定します。
[PARAM] width:
表示する幅を指定します。
[RETURN]
ブロックの実行結果を返します。
