instance method Kernel#file

file(*args) { ... } -> Rake::FileTask[permalink][rdoc] [added by rake]

ファイルタスクを定義します。

[PARAM] args:
ファイル名と依存ファイル名を指定します。

例: 

file "config.cfg" => ["config.template"] do
  open("config.cfg", "w") do |outfile|
    open("config.template") do |infile|
      while line = infile.gets
        outfile.puts line
      end
    end
  end
end
