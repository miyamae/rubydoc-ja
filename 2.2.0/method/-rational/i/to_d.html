instance method Rational#to_d

to_d(nFig) -> BigDecimal[permalink][rdoc] [added by bigdecimal/util]

自身を BigDecimal に変換します。

nFig 桁まで計算を行います。

[PARAM] nFig:
計算を行う桁数
[RETURN]
BigDecimal に変換したオブジェクト
[EXCEPTION] ArgumentError:
nFig に 0 以下を指定した場合に発生します。

例: 

require "bigdecimal"
require "bigdecimal/util"
Rational(1, 3).to_d(3).to_s  # => "0.333E0"
Rational(1, 3).to_d(10).to_s # => "0.3333333333E0"
