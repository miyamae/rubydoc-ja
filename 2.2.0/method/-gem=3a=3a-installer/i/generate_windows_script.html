instance method Gem::Installer#generate_windows_script

generate_windows_script(filename, bindir)[permalink][rdoc]

コマンドの実行を容易にするために Windows 向けのバッチファイルを作成します。

[PARAM] bindir:
実行ファイルを配置するディレクトリを指定します。
[PARAM] filename:
ファイル名を指定します。
