instance method Float#>=

self >= other -> bool[permalink][rdoc]

比較演算子。数値として等しいまたは大きいか判定します。

[PARAM] other:
比較対象の数値
[RETURN]
self よりも other の方が小さい場合か、 両者が等しい場合 true を返します。 そうでなければ false を返します。

例: 

3.14 >  3.1415  #=> false
3.14 >= 3.1415  #=> false
