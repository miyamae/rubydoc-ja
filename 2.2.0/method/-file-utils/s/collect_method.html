singleton method FileUtils.collect_method

collect_method(opt) -> Array[permalink][rdoc]

与えられたオプションを持つメソッド名の配列を返します。

[PARAM] opt:
オプション名をシンボルで指定します。 
FileUtils.collect_method(:preserve) # => ["cp", "cp_r", "copy", "install"]
