instance method BigDecimal#nonzero?

nonzero? -> self | nil[permalink][rdoc]

self が 0 以外のときに self を返します。0 のときに nil を返します。 

BigDecimal("0").nonzero? # => nil
BigDecimal("1").nonzero? # => #<BigDecimal:f7236228,'0.1E1',4(8)>
