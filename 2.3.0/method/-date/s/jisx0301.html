singleton method Date.jisx0301

jisx0301(str = '-4712-01-01', start = Date::ITALY) -> Date[permalink][rdoc]

いくつかの代表的な [[JIS:X 0301]] 書式の日付を解析し、 その情報に基づいて日付オブジェクトを生成します。

Date._jisx0301 も参照してください。

[PARAM] str:
日付をあらわす文字列
[PARAM] start:
グレゴリオ暦をつかい始めた日をあらわすユリウス日
