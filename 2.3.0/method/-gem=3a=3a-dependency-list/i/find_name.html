instance method Gem::DependencyList#find_name

find_name(full_name) -> Gem::Specification | nil[permalink][rdoc]

自身に含まれる与えられた名前を持つ Gem::Specification のインスタンスを返します。

見つからなかった場合は nil を返します。

[PARAM] full_name:
バージョンを含むフルネームで Gem の名前を指定します。

[SEE_ALSO] Gem::Specification#full_name
