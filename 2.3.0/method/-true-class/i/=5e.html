instance method TrueClass#^

self ^ other -> bool[permalink][rdoc]

other が真なら false を, 偽なら true を返します。

[PARAM] other:
排他的論理和を行なう式です。

^ は再定義可能な演算子に分類されていますので、通常は true ^ other のように使われます。 

p true ^ true     #=> false
p true ^ false    #=> true
p true ^ nil      #=> true
p true ^ (1 == 1) #=> false
p true ^ (1 + 1)  #=> false

p true.^(true)    #=> false
p true.^(false)   #=> true
p true.^(nil)     #=> true
p true.^(1 == 1)  #=> false
p true.^(1 + 1)   #=> false
