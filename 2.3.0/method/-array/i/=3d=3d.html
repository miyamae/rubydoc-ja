instance method Array#==

self == other -> bool[permalink][rdoc]

自身と other の各要素をそれぞれ順に == で比較し て、全要素が等しければ true を返します。そうでない場合には false を返します。

[PARAM] other:
自身と比較したい配列を指定します。

[SEE_ALSO] Object#== 

[ "a", "c" ]    == [ "a", "c", 7 ]     #=> false
[ "a", "c", 7 ] == [ "a", "c", 7 ]     #=> true
[ "a", "c", 7 ] == [ "a", "d", "f" ]   #=> false
