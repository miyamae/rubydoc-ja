constant OpenSSL::PKey::RSA::PKCS1_PADDING

PKCS1_PADDING -> Integer[permalink][rdoc]

PKCS #1 v1.5 で定義されているパディングモードです。

互換性目的以外での利用は推奨しません。

OpenSSL::PKey::RSA#public_encrypt、 OpenSSL::PKey::RSA#public_decrypt、 OpenSSL::PKey::RSA#private_encrypt、 OpenSSL::PKey::RSA#private_decrypt の 第二引数に指定できます。
