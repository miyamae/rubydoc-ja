module function Math.#sqrt

sqrt(x) -> Float[permalink][rdoc]

x の平方根を返します。

[PARAM] x:
正の実数
[EXCEPTION] TypeError:
xに数値以外を指定した場合に発生します。
[EXCEPTION] Math::DomainError:
x に範囲外の実数を指定した場合に発生します。
[EXCEPTION] RangeError:
xに実数以外の数値を指定した場合に発生します。
