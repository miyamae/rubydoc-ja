instance method REXML::Elements#index

index(element) -> Integer[permalink][rdoc]

element で指定した要素が何番目の子要素であるかを返します。

element が子要素でない場合は -1 を返します。

返り値は 1-origin です。

[PARAM] element:
インデックスを知りたい要素(REXML::Element オブジェクト)
