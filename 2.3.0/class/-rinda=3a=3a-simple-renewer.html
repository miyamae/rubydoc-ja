class Rinda::SimpleRenewer

クラスの継承リスト: Rinda::SimpleRenewer < DRb::DRbUndumped < Object < Kernel < BasicObject

要約

シンプルな renewer で renewer のサンプル実装です。

Rinda::TupleSpace が保持するタプルの有効期限を更新するクラスです。 このクラスはタプルの寿命をプロセスの寿命と一致させることができます。 詳しくは Rinda::TupleSpace#renewer を見てください。
