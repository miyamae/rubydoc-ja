singleton method Shell::CommandProcessor.install_system_commands

install_system_commands(prefix = "sys_") -> ()[permalink][rdoc]

全てのシステムコマンドをメソッドとして定義します。

既に定義されているコマンドを再定義することはありません。 デフォルトでは全てのコマンドに "sys_" というプレフィクスが付きます。 また、メソッド名として使用できない文字は全て "_" に置換してメソッドを定義します。 このメソッドの実行中に発生した例外は単に無視されます。

[PARAM] prefix:
プレフィクスを指定します。
