instance method WEBrick::HTTPResponse#status=

status=(status)[permalink][rdoc]

レスポンスのステータスコードを整数で指定します。 reason_phrase も適切なものに設定されます。

[PARAM] status:
ステータスコードを整数で指定します。 
require 'webrick'
res = WEBrick::HTTPResponse.new( { :HTTPVersion => "1.1" } )
res.status = 404

p res.reason_phrase    #=> "Not Found"
