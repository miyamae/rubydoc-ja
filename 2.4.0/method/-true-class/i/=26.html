instance method TrueClass#&

self & other -> bool[permalink][rdoc]

other が真なら true を, 偽なら false を返します。

[PARAM] other:
論理積を行なう式です。

& は再定義可能な演算子に分類されていますので、通常は true & other のように使われます。 

p true & true     #=> true
p true & false    #=> false
p true & nil      #=> false
p true & (1 == 1) #=> true
p true & (1 + 1)  #=> true

p true.&(true)    #=> true
p true.&(false)   #=> false
p true.&(nil)     #=> false
p true.&(1 == 1)  #=> true
p true.&(1 + 1)   #=> true
