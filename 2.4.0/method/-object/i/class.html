instance method Object#class

class -> Class[permalink][rdoc]

レシーバのクラスを返します。 

p "ruby".class #=> String
p 999999999999999.class #=> Bignum
p ARGV.class #=> Array
p self.class #=> Object
p Class.class #=> Class
p Kernel.class #=> Module
