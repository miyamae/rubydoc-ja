instance method LoadError#path

path -> String | nil[permalink][rdoc]

Kernel.#require や Kernel.#load に失敗したパスを返します。 

begin
  require 'this/file/does/not/exist'
rescue LoadError => e
  e.path # => 'this/file/does/not/exist'
end

パスが定まらない場合は nil を返します。
