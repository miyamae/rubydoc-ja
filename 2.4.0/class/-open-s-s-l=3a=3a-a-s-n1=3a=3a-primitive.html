class OpenSSL::ASN1::Primitive

クラスの継承リスト: OpenSSL::ASN1::Primitive < OpenSSL::ASN1::ASN1Data < Object < Kernel < BasicObject

要約

ASN.1 の単純型を表すクラスです。

通常はこのクラス自身は用いず、各サブクラスを利用します。

目次

インスタンスメソッド
tagging tagging=

インスタンスメソッド

tagging -> Symbol | nil[permalink][rdoc]

タグ付けの方式を返します。

:IMPLICIT、:EXPLICIT、nil のいずれかを返します。

タグ(OpenSSL::ASN1::ASN1Data#tag)が :UNIVERSAL ならば この値は無視されます。

nil は :IMPLICIT と同義です。

[SEE_ALSO] OpenSSL::ASN1::Primitive#tagging=

tagging=(tagging)[permalink][rdoc]

タグ付けの方式を設定します。

[PARAM] tagging:
タグ付けの方式(:IMPLICIT または :EXPLICIT)
