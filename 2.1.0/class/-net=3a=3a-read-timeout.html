class Net::ReadTimeout

クラスの継承リスト: Net::ReadTimeout < Timeout::Error < RuntimeError < StandardError < Exception < Object < Kernel < BasicObject

要約

データを読み出すときにタイムアウトしたときに発生する例外です。

タイムアウトまでの時間は Net::HTTP#read_timeout=、 Net::FTP#read_timeout= などで設定します。
