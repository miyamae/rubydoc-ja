module Net::HTTPExceptions

クラスの継承リスト: Net::HTTPExceptions

要約

HTTP 例外クラスです。

実際にはこれを include した以下のサブクラスの 例外が発生します。

  • Net::HTTPError
  • Net::HTTPRetriableError
  • Net::HTTPServerError
  • Net::HTTPFatalError

また、例外を発生させるためには Net::HTTPResponse#value を 呼ぶ必要があります。
