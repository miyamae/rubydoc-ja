instance method YAML::DBM#index

index(value) -> String | nil[permalink][rdoc]

value を持つ要素のキーを返します。

対応するキーが見つからなかった場合は nil を返します。

[PARAM] value:
検索したい値を指定します。YAML 形式に変換できるオブジェク トが指定できます。

[注意] 非推奨のメソッドです。代わりに #key を使用してください。
