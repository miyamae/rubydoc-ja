instance method RDoc::TopLevel#find_module_named

find_module_named(name) -> RDoc::NormalModule[permalink][rdoc]

RDoc が収集したクラスの内、name で指定した名前のモジュールを返します。 見つからなかった場合は nil を返します。

[PARAM] name:
モジュール名を文字列で指定します。
