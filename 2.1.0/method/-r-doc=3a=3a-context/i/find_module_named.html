instance method RDoc::Context#find_module_named

find_module_named(name) -> RDoc::NormalModule | nil[permalink][rdoc]

追加された RDoc::NormalModule から引数で指定した名前のモジュール を検索します。見つからなかった場合は nil を返します。

[PARAM] name:
モジュールの名前を文字列で指定します。
