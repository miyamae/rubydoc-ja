instance method StringScanner#pointer

pointer -> Fixnum[permalink][rdoc]
pos -> Fixnum

現在のスキャンポインタのインデックスを返します。

使用例 

s = StringScanner.new('test string')
s.pos         # => 0
s.scan(/\w+/) # => "test"
s.pos         # => 4
s.scan(/\w+/) # => nil
s.pos         # => 4
s.scan(/\s+/) # => " "
s.pos         # => 5
