instance method RakeFileUtils#verbose

verbose(value = nil) { ... }[permalink][rdoc]

詳細を出力するかどうかを制御します。

[PARAM] value:
真を指定すると詳細を出力します。

例 

verbose              # 現在の状態を返します。
verbose(v)           # 与えられた状態に変更します。
verbose(v) { code }  # ブロックを評価する間だけ与えられた状態に変更します。
                     # ブロックの評価が終わると元の値に戻します。
