instance method Object#must_match

must_match(regexp) -> true[permalink][rdoc] [added by minitest/spec]

自身が与えられた正規表現にマッチした場合、検査にパスしたことになります。

[PARAM] regexp:
正規表現か文字列を指定します。文字列を指定した場合は文字列そのものにマッチする 正規表現に変換してから使用します。
[EXCEPTION] MiniTest::Assertion:
自身が与えられた正規表現にマッチしなかった場合に発生します。

[SEE_ALSO] MiniTest::Assertions#assert_match
