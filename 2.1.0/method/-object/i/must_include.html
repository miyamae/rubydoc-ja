instance method Object#must_include

must_include(object) -> true[permalink][rdoc] [added by minitest/spec]

自身に与えられたオブジェクトが含まれている場合、検査にパスしたことになります。

[PARAM] object:
任意のオブジェクトを指定します。
[EXCEPTION] MiniTest::Assertion:
自身が include? メソッドを持たない場合に発生します。 自身に与えられたオブジェクトが含まれていない場合に発生します。
