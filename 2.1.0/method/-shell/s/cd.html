singleton method Shell.cd

cd(path = nil, verbose = self.verbose) -> self[permalink][rdoc]

pathをカレントディレクトリとするShellオブジェクトを生成します.

[PARAM] path:
カレントディレクトリとするディレクトリを文字列で指定します。
[PARAM] verbose:
true を指定すると冗長な出力を行います。

使用例 

require 'shell'
sh = Shell.new
sh.cd("/tmp")
