instance method String#casecmp

casecmp(other) -> Integer | nil[permalink][rdoc]

String#<=> と同様に文字列の順序を比較しますが、 アルファベットの大文字小文字の違いを無視します。

このメソッドの動作は組み込み変数 $= には影響されません。

[PARAM] other:
self と比較する文字列

例: 

p 'a' <=> 'A'       # => 1
p 'a'.casecmp('A')  # => 0
