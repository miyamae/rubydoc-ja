instance method ARGF.class#bytes

bytes { |char| ... } -> self[permalink][rdoc]
bytes -> Enumerator

このメソッドは obsolete です。 代わりに ARGF.class#each_byte を使用してください。 使用すると警告メッセージが表示されます。

self を 1 バイトずつ整数としてブロックの引数に繰り返しブロックを呼びだします。

ブロックを省略した場合には、 Enumerator オブジェクトを生成して返します。 

ARGF.bytes.to_a        #=> [104, 101, 108, 108, 111]
