instance method Matrix#row_vectors

row_vectors -> [Vector][permalink][rdoc]

自分自身を行ベクトルの配列として返します。

例: 

a1 = [ 1,  2,  3]
a2 = [10, 15, 20]
a3 = [-1, -2, 1.5]
m = Matrix[a1, a2, a3]

p m.row_vectors #=> [Vector[1, 2, 3], Vector[10, 15, 20], Vector[-1, -2, 1.5]]
