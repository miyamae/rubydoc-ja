instance method OpenStruct#[]

self[name] -> object[permalink][rdoc]

引数 name で指定した要素に対応する値を返します。

[PARAM] name:
要素の名前を文字列か Symbol オブジェクトで指定します。

例: 

person = OpenStruct.new('name' => 'John Smith', 'age' => 70)
person[:age] # => 70, person.age と同じ
