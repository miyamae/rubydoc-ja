instance method Kernel#check_sizeof

check_sizeof(type, headers = nil) -> Integer | nil[permalink][rdoc] [added by mkmf]
check_sizeof(type, headers = nil) { ... } -> Integer | nil [added by mkmf]

与えられた型のサイズを返します。

型 type がシステムに存在する場合は、グローバル変数 $defs に "-DSIZEOF_type=X" を追加し、型のサイズを返します。型 type がシステムに 存在しない場合は、nil を返します。

例えば、 

check_sizeof('mystruct') # => 12

である場合、SIZEOF_MYSTRUCT=12 というプリプロセッサマクロをコンパイラに渡します。

[PARAM] type:
検査したい型を指定します。
[PARAM] headers:
追加のヘッダファイルを指定します。
