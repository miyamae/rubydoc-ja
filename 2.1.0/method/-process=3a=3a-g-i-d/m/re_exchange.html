module function Process::GID.#re_exchange

re_exchange -> Integer[permalink][rdoc]

現在のプロセスの実グループ ID と実効グループ ID を入れ替えます。 保存ユーザ ID は新しい実効ユーザ ID と同じになります。 新しい実効グループ ID を返します。

利用できるかはプラットフォームに依存します。
