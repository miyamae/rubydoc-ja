instance method DL::CPtr#+

self + n -> DL::CPtr[permalink][rdoc]

自身のアドレスに n バイトを足した新しい CPtr オブジェクトを返します。

この返り値には、free 関数がセットされていません。

[PARAM] n:
アドレスの増分を整数で指定します。

例: 

require 'dl'

s = 'abc'
cptr = DL::CPtr[s]
p cptr[0,1]         #=> "a"
cptr += 1
p cptr[0,1]         #=> "b"
