instance method Prime::PseudoPrimeGenerator#with_object

with_object(obj) {|prime, obj| ... } -> object[permalink][rdoc]
with_object(obj) -> Enumerator

与えられた任意のオブジェクトと要素をブロックに渡して評価します。

[PARAM] obj:
任意のオブジェクトを指定します。
[RETURN]
最初に与えられたオブジェクトを返します。
[RETURN]
ブロックを与えられた場合は obj を返します。ブロックを与えられなかった場合は Enumerator を返します。
