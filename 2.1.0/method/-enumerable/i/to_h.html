instance method Enumerable#to_h

to_h(*args) -> Hash[permalink][rdoc]

self を [key, value] のペアの配列として解析した結果を Hash にして 返します。ペアにならない要素は無視されます。

[PARAM] args:
each の呼び出し時に引数として渡されます。 
%i[hello world].each_with_index.to_h
  # => {:hello => 0, :world => 1}
