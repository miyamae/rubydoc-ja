library irb/slex

要約

Ruby のソースコードを字句解析するためのサブライブラリです。 irb/ruby-lex から使用されます。

このライブラリで定義されているメソッドはユーザが直接使用するものではありません。

同時にrequireされるライブラリ

e2mmap

例外クラスに特定のエラーメッセージ用フォーマットを関連づけるためのライブラリです。

irb/notifier

ライブラリ内部で使用します。

irb/output-method

irb が出力を扱うためのサブライブラリです。
