module function Kernel.#binding

binding -> Binding[permalink][rdoc]

変数・メソッドなどの環境情報を含んだ Binding オブジェクトを 生成して返します。通常、Kernel.#eval の第二引数として使います。 

def foo
  a = 1
  binding
end

eval("p a", foo)  #=> 1
