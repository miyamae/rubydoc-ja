instance method DL::CFunc#calltype

calltype -> Symbol[permalink][rdoc]
calltype=(sym)

自身の呼出規約を表すアクセサです。 サポートされている規約は :cdecl と :stdcall です。

[PARAM] sym:
自身の呼出規約を Symbol オブジェクトで指定します。
