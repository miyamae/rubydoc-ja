instance method Symbol#casecmp

casecmp(other) -> -1 | 0 | 1[permalink][rdoc]

Symbol#<=> と同様にシンボルに対応する文字列の順序を比較しますが、 アルファベットの大文字小文字の違いを無視します。

[PARAM] other:
比較対象のシンボルを指定します。 
p :a <=> :A      # => 1
p :a.casecmp(:A) # => 0
