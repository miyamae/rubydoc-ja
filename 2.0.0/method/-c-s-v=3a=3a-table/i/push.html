instance method CSV::Table#push

push(*rows) -> self[permalink][rdoc]

複数の行を追加するためのショートカットです。

[PARAM] rows:
CSV::Rows のインスタンスか配列を指定します。

以下と同じです。 

csv = CSV.new("a,b,c\n,1,2,3", headers: true)
table = csv.read
rows = [[4, 5, 6], [7, 8, 9]]
rows.each{|row| table << row }
