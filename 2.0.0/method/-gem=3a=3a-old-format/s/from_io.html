singleton method Gem::OldFormat.from_io

from_io(io, gem_path = '(io)') -> Gem::OldFormat[permalink][rdoc]

Gem ファイルからデータを読み込んだ IO を受け取り、自身を初期化して返します。

[PARAM] io:
Gem パッケージの内容を読み込んだ IO オブジェクトを指定します。
[PARAM] gem_path:
Gem ファイルのパスを指定します。
