instance method Net::HTTP#copy

copy(path, initheader = nil) -> Net::HTTPResponse[permalink][rdoc]

サーバの path に COPY リクエストを ヘッダを initheader として送ります。

レスポンスを Net::HTTPResponse のオブジェクト で返します。

[PARAM] path:
リクエストを送るパスを文字列で与えます。
[PARAM] initheader:
リクエストのヘッダを「文字列=>文字列」の ハッシュで与えます。
