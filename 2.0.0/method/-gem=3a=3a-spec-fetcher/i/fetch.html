instance method Gem::SpecFetcher#fetch

fetch(dependency, all = false, matching_platform = true) -> Array[permalink][rdoc]

依存関係を満たす gemspec の配列を返します。

[PARAM] dependency:
依存関係を指定します。
[PARAM] all:
真を指定するとマッチする全てのバージョンの情報を返します。
[PARAM] matching_platform:
偽を指定すると全てのプラットフォームの情報を返します。

[SEE_ALSO] Gem::Dependency
