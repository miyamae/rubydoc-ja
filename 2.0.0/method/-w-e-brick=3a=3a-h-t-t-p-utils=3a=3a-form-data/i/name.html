instance method WEBrick::HTTPUtils::FormData#name

name -> String | nil[permalink][rdoc]
name=(value)

フォームデータの name 属性を文字列で表すアクセサです。

[PARAM] value:
フォームデータの name 属性を文字列で指定します。

例: 

require "webrick/cgi"
class MyCGI < WEBrick::CGI
  def do_GET(req, res)
    p req.query['q'].name   #=> "q"
  end
end
MyCGI.new.start()
