instance method Curses::Window#subwin

subwin(height, width, y, x) -> Curses::Window[permalink][rdoc]

画面の座標 (x,y) を左上端とし、幅 width 高さ height とするサブウィンドウを作成し、 それを表現する Curses::Window オブジェクトを返します。

[PARAM] height:
ウィンドウの高さを指定します。
[PARAM] width:
ウィンドウの幅を指定します。
[PARAM] y:
ウィンドウの左上端の y 座標を指定します。
[PARAM] x:
ウィンドウの左上端の x 座標を指定します。
