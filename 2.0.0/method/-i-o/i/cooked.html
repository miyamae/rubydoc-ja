instance method IO#cooked

cooked {|io| ... } -> object [added by io/console]

cooked モードを有効にして指定されたブロックを評価します。

ブロック引数には self が渡されます。ブロックを評価した結果を返します。

[EXCEPTION] LocalJumpError:
ブロックを指定しなかった場合に発生します。

以下の例では、標準入力からエコーバック付きで文字列を一行読み込みます。 

require "io/console"
STDIN.cooked(&:gets)
