instance method DRb::DRbServer#verbose=

verbose=(on)[permalink][rdoc]

サーバの verbose mode を真偽値で設定します。

verbose mode が on の場合は失敗したメソッド呼出のログが標準出力に出力 されます。

[PARAM] on:
真を渡すと verbose mode が on になります
