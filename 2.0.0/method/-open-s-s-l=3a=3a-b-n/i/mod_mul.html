instance method OpenSSL::BN#mod_mul

mod_mul(other, m) -> OpenSSL::BN[permalink][rdoc]

(self * other) % m を返します。

例: 

BN.new("7").mod_mul(BN.new("3"), BN.new("6")) # => 3
[PARAM] other:
積を取る数
[PARAM] m:
剰余を取る数
[EXCEPTION] OpenSSL::BNError:
計算時エラー
