instance method Numeric#to_c

to_c -> Complex[permalink][rdoc]

自身を複素数 (Complex) に変換します。Complex(self, 0) を返します。

例: 

1.to_c              # => (1+0i)
-1.to_c             # => (-1+0i)
1.0.to_c            # => (1.0+0i)
Rational(1, 2).to_c # => ((1/2)+0i)

Numeric のサブクラスは、このメソッドを適切に再定義しなければなりません。
