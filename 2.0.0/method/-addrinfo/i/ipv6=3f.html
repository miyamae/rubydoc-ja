instance method Addrinfo#ipv6?

ipv6? -> bool[permalink][rdoc]

アドレスが IPv6 のものならば true を返します。 

Addrinfo.tcp("127.0.0.1", 80).ipv6? #=> false
Addrinfo.tcp("::1", 80).ipv6?       #=> true
Addrinfo.unix("/tmp/sock").ipv6?    #=> false
