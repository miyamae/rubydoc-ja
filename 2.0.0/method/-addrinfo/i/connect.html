instance method Addrinfo#connect

connect -> Socket[permalink][rdoc]
connect {|sock| ... } -> object

自身のアドレスへソケットを接続します。

ブロックが渡されたときにはそのブロックに接続済み Socket オブジェクトが渡されます。ブロックの返り値がメソッドの返り値となります。 ブロックを省略した場合は、接続済み Socket オブジェクトが返されます。
