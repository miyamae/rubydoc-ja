module function FileTest.#world_writable?

world_writable?(path) -> bool[permalink][rdoc]

path が全てのユーザから書き込めるならば、そのファイルのパーミッションを表す 整数を返します。そうでない場合は nil を返します。

整数の意味はプラットフォームに依存します。

[PARAM] path:
パスを表す文字列を指定します。 
m = FileTest.world_writable?("/tmp")
"%o" % m                               #=> "777"
