instance method File::Stat#ftype

ftype -> String[permalink][rdoc]

ファイルのタイプを表す文字列を返します。

文字列は以下のうちのいずれかです。 

"file"
"directory"
"characterSpecial"
"blockSpecial"
"fifo"
"link"
"socket"

"unknown"

サンプル 

fs = File::Stat.new($0)
p fs.ftype #=> "file"
p File::Stat.new($:[0]).ftype #=> "directory"
