instance method Time#to_json

to_json(*args) -> String[permalink][rdoc] [added by json/add/core]

自身を JSON 形式の文字列に変換して返します。

内部的にはハッシュにデータをセットしてから Hash#to_json を呼び出しています。

[PARAM] args:
引数はそのまま Hash#to_json に渡されます。
