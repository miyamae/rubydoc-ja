module function DL.#realloc

realloc(addr, size) -> Integer[permalink][rdoc]

addr で指定したメモリ領域を size バイトにリサイズし、その領域を指す整数 を返します。

addr には DL.malloc で確保したメモリ領域を渡します。 また、リサイズの結果、返り値が addr と異なる場合があります。

[PARAM] addr:
リサイズしたいメモリアドレス整数
[PARAM] size:
リサイズ後のバイト数

[SEE_ALSO] DL.malloc
