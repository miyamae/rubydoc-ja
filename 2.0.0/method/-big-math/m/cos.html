module function BigMath.#cos

cos(x, prec) -> BigDecimal[permalink][rdoc] [added by bigdecimal/math]

x の余弦関数を prec で指定した精度で計算します。単位はラジアンです。x に無限大や NaN を指定した場合には NaN を返します。

[PARAM] x:
計算対象の BigDecimal オブジェクト。単位はラジアン。
[PARAM] prec:
計算結果の精度。
[EXCEPTION] ArgumentError:
prec に 0 以下が指定された場合に発生します。

例: 

require "bigdecimal/math"

puts BigMath::cos(BigDecimal.new('0.5'), 10) #=> 0.8775825618903727161162815826327690580439923E0
