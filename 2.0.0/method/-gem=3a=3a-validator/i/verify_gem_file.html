instance method Gem::Validator#verify_gem_file

verify_gem_file(gem_path) -> ()[permalink][rdoc]

与えられた Gem ファイルのパスを使用して MD5 チェックサムを検証します。

[PARAM] gem_path:
Gem ファイルのパスを指定します。
[EXCEPTION] Gem::VerificationError:
Gem ファイルが見つからなかった場合に発生します。
