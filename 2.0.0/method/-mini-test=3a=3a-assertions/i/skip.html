instance method MiniTest::Assertions#skip

skip(message = nil, backtrace = caller)[permalink][rdoc]

このメソッドを呼び出したテストメソッドをスキップします。

[PARAM] message:
メッセージを指定します。
[PARAM] backtrace:
例外発生時のスタックトレースで、Kernel.#caller の戻り値と同じ 形式で指定しなければいけません。
[EXCEPTION] MiniTest::Skip:
必ず発生します。
