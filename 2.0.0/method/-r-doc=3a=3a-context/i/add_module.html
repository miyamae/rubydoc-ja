instance method RDoc::Context#add_module

add_module(class_type, name) -> RDoc::NormalModule | RDoc::SingleClass | RDoc::NormalClass[permalink][rdoc]

引数で指定したモジュールを追加します。

ただし、クラスとして既に同名のものが登録されていた場合は、何もせずにそ のクラスを返します。

[PARAM] class_type:
追加するクラス、モジュールを RDoc::NormalClass、 RDoc::SingleClass、RDoc::NormalModule オ ブジェクトのいずれかで指定します。
[PARAM] name:
クラス名を文字列で指定します。
