module function Zlib.#adler32_combine

adler32_combine(adler1, adler2, length) -> Integer[permalink][rdoc]

[PARAM] adler1:
Adler-32 チェックサムを指定します。
[PARAM] adler2:
Adler-32 チェックサムを指定します。
[PARAM] length:
