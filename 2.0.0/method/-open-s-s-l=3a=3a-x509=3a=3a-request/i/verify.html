instance method OpenSSL::X509::Request#verify

verify(key) -> bool[permalink][rdoc]

署名を検証します。

検証に成功した場合には true を、失敗した場合には false を返します。

[PARAM] key:
検証に利用する公開鍵(OpenSSL::PKey::PKey のサブクラスのインスタンス)
[EXCEPTION] OpenSSL::X509::RequestError:
検証時にエラーが生じた場合に発生します
