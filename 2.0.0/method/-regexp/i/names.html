instance method Regexp#names

names -> [String][permalink][rdoc]

正規表現に含まれる名前付きキャプチャ(named capture)の名前を 文字列の配列で返します。 

/(?<foo>.)(?<bar>.)(?<baz>.)/.names

#=> ["foo", "bar", "baz"]
   /(?<foo>.)(?<foo>.)/.names
#=> ["foo"]

/(.)(.)/.names
#=> []
