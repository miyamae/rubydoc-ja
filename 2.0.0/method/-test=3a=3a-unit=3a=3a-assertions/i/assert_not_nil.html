instance method Test::Unit::Assertions#assert_not_nil

assert_not_nil(object, message = "") -> ()[permalink][rdoc]

object が nil でないならばパスします。

[PARAM] object:
検証するオブジェクトを指定します。
[PARAM] message:
assert が失敗した時に表示するメッセージを文字列で指定し ます。指定しなかった場合は表示しません。
[EXCEPTION] MiniTest::Assertion:
assert が失敗した時に発生します。
