singleton method OpenSSL::ASN1::UTF8String.new

new(value) -> OpenSSL::ASN1::UTF8String[permalink][rdoc]
new(value, tag, tagging, tag_class) -> OpenSSL::ASN1::UTF8String

ASN.1 の UTF8String 型の値を表現する OpenSSL::ASN1::UTF8String オブジェクトを 生成します。

value 以外の引数を省略した場合はタグクラスは :UNIVERSAL、 タグ は OpenSSL::ASN1::UTF8STRING となります。

[PARAM] value:
ASN.1 値を表す Ruby のオブジェクト(文字列)
[PARAM] tag:
タグ番号
[PARAM] tagging:
タグ付けの方法(:IMPLICIT もしくは :EXPLICIT)
[PARAM] tag_class:
タグクラス(:UNIVERSAL, :CONTEXT_SPECIFIC, :APPLICATION, :PRIVATE のいずれか)
