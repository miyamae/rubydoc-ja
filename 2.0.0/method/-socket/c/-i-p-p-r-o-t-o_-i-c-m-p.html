constant Socket::IPPROTO_ICMP

IPPROTO_ICMP -> Integer[permalink][rdoc]

Control message protocol。 BasicSocket#getsockopt, BasicSocket#setsockopt の level 引数に使用します。

また、Socket.open の protocol 引数に渡す利用法もあります。
