instance method Array#shuffle!

shuffle! -> self[permalink][rdoc]
shuffle!(random: rng) -> self

配列を破壊的にランダムシャッフルします。

[PARAM] rng:
乱数生成器(主に Random オブジェクト)を指定します。 選択する要素のインデックスを返す rand メソッドに応答するオブ ジェクトであれば指定する事ができます。rand メソッドの引数に は Random#rand(max) のように選択可能なインデックスの最 大値が指定されます。 Kernel.#rand、Random を使用しないオブジェクトを 指定した場合、Kernel.#srandの指定に影響されません。

例: 

a = [ 1, 2, 3 ]           #=> [1, 2, 3]
a.shuffle!                #=> [2, 3, 1]
a                         #=> [2, 3, 1]
