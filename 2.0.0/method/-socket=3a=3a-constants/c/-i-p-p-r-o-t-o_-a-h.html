constant Socket::Constants::IPPROTO_AH

IPPROTO_AH -> Integer[permalink][rdoc]

IPv6 auth header。 BasicSocket#getsockopt, BasicSocket#setsockopt の level 引数に使用します。

また、Socket.open の protocol 引数に渡す利用法もあります。
