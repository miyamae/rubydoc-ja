constant Socket::Constants::IPPROTO_NONE

IPPROTO_NONE -> Integer[permalink][rdoc]

IP6 no next header。 BasicSocket#getsockopt, BasicSocket#setsockopt の level 引数に使用します。

また、Socket.open の protocol 引数に渡す利用法もあります。
