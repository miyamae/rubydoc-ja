singleton method IRB::MagicFile.open

open(path) -> File[permalink][rdoc]
open(path) { |io| ... } -> object

引数 path で指定したファイルを開いて、ファイル中のマジックコメントをエ ンコーディングに設定します。

ブロックを指定した場合はブロックの実行結果を返します。ブロックを指定し なかった場合はエンコーディングが設定された File オブジェクトを返 します。

[PARAM] path:
パスを文字列で指定します。
