instance method Module#public_instance_methods

public_instance_methods(inherited_too = true) -> [Symbol][permalink][rdoc]

そのモジュールで定義されている public メソッド名 の一覧を配列で返します。

[PARAM] inherited_too:
false を指定するとそのモジュールで定義されているメソッドのみ返します。
