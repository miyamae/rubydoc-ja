instance method Module#private_constant

private_constant(*name) -> self

name で指定した定数の可視性を private に変更します。

[PARAM] name:
0 個以上の String か Symbol を指定します。
[EXCEPTION] NameError:
存在しない定数を指定した場合に発生します。
[EXCEPTION] SecurityError:
セーフレベル($SAFE)が 4 以上で自身が untrust なオブジェクトではない場合に発生します。
[RETURN]
自身を返します。

[SEE_ALSO] Module#public_constant, Object#untrusted?
