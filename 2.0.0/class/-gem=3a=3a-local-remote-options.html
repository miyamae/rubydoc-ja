module Gem::LocalRemoteOptions

クラスの継承リスト: Gem::LocalRemoteOptions

要約

Gem::Command で使用する --local, --remote オプションのためのモジュールです。

目次

インスタンスメソッド
accept_uri_http add_bulk_threshold_option add_local_remote_options add_proxy_option add_source_option add_update_sources_option both? local? remote?

インスタンスメソッド

accept_uri_http[permalink][rdoc]

HTTP の URI を扱えるようにするためのメソッドです。

add_bulk_threshold_option[permalink][rdoc]

オプション --bulk-threshold を追加します。

add_local_remote_options[permalink][rdoc]

オプション --local, --remote, --both を追加します。

add_proxy_option[permalink][rdoc]

オプション --http-proxy を追加します。

add_source_option[permalink][rdoc]

オプション --source を追加します。

add_update_sources_option[permalink][rdoc]

オプション --update-source を追加します。

both? -> bool[permalink][rdoc]

ローカルとリモートの情報を両方とも取得する場合は真を返します。 そうでない場合は偽を返します。

local? -> bool[permalink][rdoc]

ローカルの情報を取得する場合は真を返します。 そうでない場合は偽を返します。

remote? -> bool[permalink][rdoc]

リモートの情報を取得する場合は真を返します。 そうでない場合は偽を返します。
