class Rinda::TupleSpaceProxy

クラスの継承リスト: Rinda::TupleSpaceProxy < Object < Kernel < BasicObject

要約

リモートの Rinda::TupleSpace オブジェクトを包む プロクシクラスです。

Rinda::TupleSpace#take でタプルの受け渡し時にタプルが 消失する可能性を下げるためのプロクシクラスです。
