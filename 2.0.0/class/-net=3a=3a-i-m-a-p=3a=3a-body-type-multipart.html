class Net::IMAP::BodyTypeMultipart

クラスの継承リスト: Net::IMAP::BodyTypeMultipart < Struct < Enumerable < Object < Kernel < BasicObject

要約

マルチパートなメッセージを表すクラスです。

詳しくは MIME のRFC([RFC2045])を参照してください。
