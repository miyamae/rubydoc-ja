module Syslog::Constants

クラスの継承リスト: Syslog::Constants < Syslog::Option < Syslog::Facility < Syslog::Level < Syslog::Macros

要約

このモジュールにはシステムで使用可能なLOG_*定数、モジュール関数が定義さ れています。 

例:
  require 'syslog'
  include Syslog::Constants

それぞれの定数、モジュール関数は以下のモジュールに分けて定義されています。

  • Syslog::Option
  • Syslog::Facility
  • Syslog::Level
  • Syslog::Macros
