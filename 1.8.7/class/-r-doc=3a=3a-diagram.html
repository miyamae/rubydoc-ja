class RDoc::Diagram

クラスの継承リスト: RDoc::Diagram < Object < Kernel

要約

モジュールやクラスの関係を描く図を作成するためのクラスです。

特異メソッド

new(info, options) -> RDoc::Diagram[permalink][rdoc]

自身を初期化します。

[PARAM] info:
RDoc::TopLevel オブジェクトの配列を指定します。
[PARAM] options:
Options オブジェクトを指定します。

インスタンスメソッド

draw -> ()[permalink][rdoc]

自身が持つ全ての RDoc::TopLevel オブジェクトを解析してモジュール やクラスの関係を描く図を作成します。

定数

DOT_PATH -> "dot"[permalink][rdoc]

画像を作成するサブディレクトリ名を返します。

FONT -> "Arial"[permalink][rdoc]

RDoc::Diagram が使用するフォント名を返します。
