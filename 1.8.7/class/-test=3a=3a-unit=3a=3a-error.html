class Test::Unit::Error

クラスの継承リスト: Test::Unit::Error < Object < Kernel

要約

テスト時のエラーを表現するクラスです。テスト中に例外が発生した時に Test::Unit::TestCase から作られます。

目次

インスタンスメソッド

特異メソッド

new(test_name, exception) -> Test::Unit::Error[permalink][rdoc]

Test::Unit::Error オブジェクトのインスタンスを生成します。

[PARAM] test_name:
対応するテストメソッドの名前を指定します。
[PARAM] exception:
対応する例外オブジェクトを指定します。

インスタンスメソッド

exception -> Exception[permalink][rdoc]

自身に対応する例外オブジェクトを返します。

long_display -> String[permalink][rdoc]
to_s -> String

エラーの詳細な説明を文字列で返します。

message -> String[permalink][rdoc]

発生した例外に関連するエラーメッセージを文字列で返します。

Test::Unit::Error#short_display や Test::Unit::Error#long_display で使われます。

[SEE_ALSO] Test::Unit::Error#short_display, Test::Unit::Error#long_display

short_display -> String[permalink][rdoc]

エラーの簡単な説明を文字列で返します。

single_character_display -> String[permalink][rdoc]

テストメソッド実行中に例外が発生した時に表示する 'E' を返します。

test_name -> String[permalink][rdoc]

自身に対応するテストメソッドの名前を返します。
