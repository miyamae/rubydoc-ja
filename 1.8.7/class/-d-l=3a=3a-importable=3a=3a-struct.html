types には DL::Types のインスタンス、contents には構造体の定義を 以下のようにして与えます。 

["int size",
 "char *str"]

このようにして生成されたStructオブジェクトは、以下の構造体のように振舞います。 

struct {
  int size;
  char *str;
}
