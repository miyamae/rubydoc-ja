module Syslog::Constants

クラスの継承リスト: Syslog::Constants

要約

このモジュールにはシステムで使用可能なLOG_*定数、モジュール関数が定義さ れています。 

例:
  require 'syslog'
  include Syslog::Constants

モジュール関数

LOG_MASK(priority) -> Fixnum[permalink][rdoc]

1つの優先度に対するマスクを作成します。

[PARAM] priority:
priority は優先度を示す定数を指定します。 詳しくは、Syslog::Constantsを参照してください。

例: 

Syslog.mask = Syslog::LOG_MASK(Syslog::LOG_ERR)
LOG_UPTO(priority) -> Fixnum[permalink][rdoc]

priorityまでのすべての優先度のマスクを作成します。

[PARAM] priority:
priority は優先度を示す定数を指定します。 詳しくは、Syslog::Constantsを参照してください。

例: 

Syslog.mask = Syslog::LOG_UPTO(Syslog::LOG_ERR)
