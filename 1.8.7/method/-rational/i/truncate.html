instance method Rational#to_i

to_i -> Integer[permalink][rdoc]
truncate -> Integer

小数点以下を切り捨てて値を整数に変換します。

例: 

Rational(2, 3).to_i   # => 0
Rational(3).to_i      # => 3
Rational(300.6).to_i  # => 300
Rational(98, 71).to_i # => 1
Rational(-30, 2).to_i # => -15
