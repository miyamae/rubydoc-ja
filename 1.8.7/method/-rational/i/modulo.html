instance method Rational#%

self % other -> Rational | Float[permalink][rdoc]
modulo(other) -> Rational | Float

剰余を計算します。

[PARAM] other:
自身を割る数

other に Float を指定した場合は、計算結果を Float で返しま す。

例: 

r = Rational(7, 4)
r % Rational(1, 2)   # => (1/4)
r % 1                # => (3/4)
r % Rational(1, 7)   # => (1/28)
r % 0.26             # => 0.19
[EXCEPTION] ZeroDivisionError:
other が 0 の時に発生します。
