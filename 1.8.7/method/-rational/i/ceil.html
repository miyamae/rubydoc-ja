instance method Rational#ceil

ceil -> Integer[permalink][rdoc]

自身と等しいかより大きな整数のうち最小のものを返します。

例: 

Rational(3).ceil      # => 3
Rational(2, 3).ceil   # => 1
Rational(-3, 2).ceil  # => -1

[SEE_ALSO] Rational#floor, Rational#round, Rational#truncate
