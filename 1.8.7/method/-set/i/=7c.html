instance method Set#+

union(enum) -> Set[permalink][rdoc]
self + enum -> Set
self | enum -> Set

和集合、すなわち、2 つの集合の少なくともどちらか一方に属するすべての 要素からなる新しい集合を作ります。

[PARAM] enum:
Enumerable オブジェクトを指定します。
[EXCEPTION] ArgumentError:
引数が Enumerable オブジェクトでない場合に発生します。 
p Set[10, 20, 30] + Set[10, 20, 40]
#=> #<Set: {40, 30, 20, 10}>
