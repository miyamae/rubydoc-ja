instance method Shell#install

install(from, to, mode = nil, verbose = false) -> ()[permalink][rdoc]

File クラスにある同名のクラスメソッドと同じです.

[PARAM] from:
コピー元のファイル。
[PARAM] to:
コピー先のファイル。
[PARAM] mode:
ファイルのアクセスモード。8進数で指定します。
[PARAM] verbose:
真を指定すると詳細を表示します。
