instance method String#jlength

jlength -> Integer [added by jcode]
jsize -> Integer [added by jcode]

String#length の日本語対応版です。

文字数を整数で返します。

例： 

#!/usr/bin/env ruby

$KCODE = 'EUC'
zstr = 'ＡＢＣＤＥＦ'
hogehoge = 'hogehoge'

p zstr.size       # => 12   これは正しくない
p hogehoge.size   # => 8

require 'jcode'
p zstr.jsize      # => 6    これは正しい
p hogehoge.jsize  # => 8
