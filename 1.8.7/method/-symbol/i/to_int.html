instance method Symbol#to_i

to_i -> Integer[permalink][rdoc]
to_int -> Integer

シンボルに対応する整数を返します。

このメソッドで得られる整数は、内部実装で名前の管理に使われています。 オブジェクトに対応する整数( Object#object_id で得ら れます)と Symbol に対応する整数は別のものです。

逆にこの整数から対応するシンボルを得るには Fixnum#to_sym が使えます 

id = :foo.to_i
p id                  #=> 8881
p id.to_sym           #=> :foo
