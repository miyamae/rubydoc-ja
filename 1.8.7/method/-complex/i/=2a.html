instance method Complex#*

self * c -> Complex[permalink][rdoc]

複素数 c を乗じた結果を返します。

[PARAM] c:
乗算する数
[RETURN]
乗算結果をComplexクラスのオブジェクトとして返します。

例: 

c =  Complex(1, 1)    #=> Complex(1, 1)
p c * Complex(2, 2)   #=> Complex(0, 4)
p c * 3               #=> Complex(3, 3)
