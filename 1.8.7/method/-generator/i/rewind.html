instance method Generator#rewind

rewind -> self[permalink][rdoc]

ジェネレータを最初に巻き戻します。self を返します。

例: 

g = Generator.new() do |g|
      ['A', 'B', 'C', 'Z'].each{|s|
        g.yield s
      }
    end
p g.next # => 'A'
p g.next # => 'B'
g.rewind
p g.next # => 'A'
