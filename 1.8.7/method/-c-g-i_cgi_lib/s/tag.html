singleton method CGI_cgi_lib.tag

tag(element, attributes = {}) -> String[permalink][rdoc]
tag(element, attributes = {}) { ... } -> String

HTML タグを表す文字列を作って返します。

ブロックが与えられた場合は、ブロックを評価した結果が HTML タグに挟まれます。

[PARAM] element:
タグの名前を指定します。
[PARAM] attributes:
タグに指定する属性をハッシュで指定します。
