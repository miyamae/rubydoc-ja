module function Math.#atan2

atan2(y, x) -> Float[permalink][rdoc]

y / x の逆正接関数の値を返します。

[PARAM] y:
実数
[PARAM] x:
実数
[RETURN]
返される値の範囲は [-π, π] です。 
Math.atan2(1,0)   #=>  1.5707963267949
Math.atan2(-1,0)  #=> -1.5707963267949
atan2(z) -> Complex[permalink][rdoc] [redefined by complex]

複素関数としてのatan2関数の結果を返します。

atan2関数は以下で定義されます。 

atan2(y, x) = -i * log( (x + i * y) / sqrt( x*x + y*y ) )
[PARAM] z:
複素数
