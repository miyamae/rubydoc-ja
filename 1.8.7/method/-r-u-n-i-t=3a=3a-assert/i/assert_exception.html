instance method RUNIT::Assert#assert_exception

assert_exception(exception, message = "") { ... } -> ()[permalink][rdoc]

与えられたブロックを評価中に与えられた例外が発生する場合、検査にパスしたことになります。

[PARAM] exception:
例外クラスを指定します。
[PARAM] message:
検査に失敗したときのメッセージを指定します。
