instance method Array#each

each {|item| .... } -> self[permalink][rdoc]
each -> Enumerable::Enumerator

各要素に対してブロックを評価します。

ブロックが与えられなかった場合は、自身と each から生成した Enumerable::Enumerator オブジェクトを返します。 

[1, 2, 3].each do |i|
  puts i
end
#=> 1
    2
    3
