instance method Array#nitems

nitems -> Integer[permalink][rdoc]
nitems {|obj| ... } -> Integer

nil でない要素の数を返します。

ブロックが与えられた場合は、各要素を引数として評価し nil でない値を返した 要素の数を返します。

例: 

p [1, nil, 3, nil].nitems              #=> 2
p [1, nil, 3, nil].nitems{|e| e == 1}  #=> 2
