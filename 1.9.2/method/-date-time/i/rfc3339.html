instance method DateTime#iso8601

iso8601(n = 0) -> String[permalink][rdoc]
rfc3339(n = 0) -> String
xmlschema(n = 0) -> String

[[ISO:8601]] 書式の文字列を返します (XML Schema の dateTime 相当)。 省略可能な引数により、印字する秒の小数点以下の桁数を与えることができます。

[PARAM] n:
小数点以下の桁数
