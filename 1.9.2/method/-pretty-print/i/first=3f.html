instance method PrettyPrint#first?

first? -> bool[permalink][rdoc]

このメソッドは obsolete です。

現在のグループで first? に対する最初の呼び出しかどうかを判定する 述語です。これはカンマで区切られた値を整形するのに有用です。 

pp.group(1, '[', ']') {
  xxx.each {|yyy|
    unless pp.first?
      pp.text ','
      pp.breakable
    end
    ... pretty printing yyy ...
  }
}
