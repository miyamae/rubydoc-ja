instance method Numeric#rect

rect -> [Numeric, Numeric][permalink][rdoc]
rectangular -> [Numeric, Numeric]

[self, 0] を返します。

例: 

1.rect    # => [1, 0]
-1.rect   # => [-1, 0]
1.0.rect  # => [1.0, 0]
-1.0.rect # => [-1.0, 0]

Numeric のサブクラスは、このメソッドを適切に再定義しなければなりません。
