instance method Hash#default=

default=(value)[permalink][rdoc]

ハッシュのデフォルト値を value に変更します。対応する値が存 在しないキーで検索した時にはこの値を返すようになります。

デフォルト値（ブロックを含む）が既に設定してあった場合も value で上書きします。

[PARAM] value:
設定するデフォルト値です。
[RETURN]
value を返します。 
h = {}
p h.default #=>nil

h.default = "default"
p h.default #=>"default"
