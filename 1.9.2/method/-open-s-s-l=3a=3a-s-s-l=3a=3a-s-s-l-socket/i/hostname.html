instance method OpenSSL::SSL::SSLSocket#hostname

hostname -> String | nil

TLS の Server Name Indication 拡張で利用するサーバのホスト名を返します。

OpenSSL::SSL::SSLSocket#hostname= で設定した値がそのまま返されます。

設定していない場合は nil を返します。
