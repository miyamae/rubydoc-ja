instance method Vector#collect

collect {|x| ... } -> Vector[permalink][rdoc]
map {|x| ... } -> Vector
collect -> Enumerator
map -> Enumerator

ベクトルの各要素に対してブロックを評価した結果を、要素として持つベクトルを生成します。

ブロックを省略した場合は Enumerator を返します。

例: 

a = [1, 2, 3.5, -10]
v1 = Vector.elements(a)
p v1 #=> Vector[1, 2, 3.5, -10]
v2 =  v1.map{|x|
  x * -1
}
p v2 #=> Vector[-1, -2, -3.5, 10]
