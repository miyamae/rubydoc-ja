instance method StringIO#closed?

closed? -> bool[permalink][rdoc]

自身が既に close されていた場合に true を返します。そうでない場合は、false を返します。 

sio = StringIO.open("hoge")
p sio.closed? # => false
sio.close_read
p sio.closed? # => false
sio.close_write
sio.closed?   # => true
