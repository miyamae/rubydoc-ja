instance method BigDecimal#zero?

zero? -> bool[permalink][rdoc]

self が 0 のときに true を返します。それ以外のときに false を返します。 


require "bigdecimal"
BigDecimal("0").zero? # => true
BigDecimal("1").zero? # => false
