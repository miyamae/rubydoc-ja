instance method BigDecimal#save_rounding_mode

save_rounding_mode { ... } -> object[permalink][rdoc]

丸め処理に関する BigDecimal.mode の設定を保存してブロックを評価します。ブロック中で変更した設定はブロックの評価後に復元されます。

ブロックの評価結果を返します。
