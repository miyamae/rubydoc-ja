instance method String#capitalize

capitalize(*options) -> String[permalink][rdoc]

文字列先頭の文字を大文字に、残りを小文字に変更した文字列を返します。

[PARAM] options:
オプションの詳細は String#downcase を参照してください。 
例

p "foobar--".capitalize   # => "Foobar--"
p "fooBAR--".capitalize   # => "Foobar--"
p "FOOBAR--".capitalize   # => "Foobar--"
