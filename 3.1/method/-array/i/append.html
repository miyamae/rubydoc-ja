instance method Array#append

push(*obj) -> self[permalink][rdoc]
append(*obj) -> self

指定された obj を順番に配列の末尾に追加します。引数を指定しなければ何もしません。

[PARAM] obj:
自身に追加したいオブジェクトを指定します。 
例

array = [1, 2, 3]
array.push 4
array.push [5, 6]
array.push 7, 8
p array          # => [1, 2, 3, 4, [5, 6], 7, 8]
