instance method Numeric#+@

+ self -> self[permalink][rdoc]

単項演算子の + です。 self を返します。 

例

+ 10            # => 10
+ (-10)           # => -10
+ 0.1           # => 0.1
+ (3r) # => (3/1)
+ (1+3i)        # => (1+3i)
