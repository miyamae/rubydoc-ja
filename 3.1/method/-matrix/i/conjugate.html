instance method Matrix#conj

conjugate -> Matrix[permalink][rdoc]
conj -> Matrix

複素共役を取った行列を返します。 

例

require 'matrix'
Matrix[[Complex(1,2), Complex(0,1), 0], [1, 2, 3]]
  # => 1+2i   i  0
  #       1   2  3
Matrix[[Complex(1,2), Complex(0,1), 0], [1, 2, 3]].conjugate
  # => 1-2i  -i  0
  #       1   2  3
