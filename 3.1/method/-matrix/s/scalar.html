singleton method Matrix.scalar

scalar(n, value) -> Matrix[permalink][rdoc]

対角要素が全てvalue(数)で、非対角要素が全て0であるようなn次の正方行列を生成します。

[PARAM] n:
生成する行列の次元
[PARAM] value:
生成する行列の対角要素の値 
例

require 'matrix'

m = Matrix.scalar(3, 2.5)
p m # => Matrix[[2.5, 0, 0], [0, 2.5, 0], [0, 0, 2.5]]
