constant Random::DEFAULT

DEFAULT -> Class[permalink][rdoc]

Ruby 3.0 から非推奨です。代わりに Random クラスオブジェクトを擬似乱数生成器として使用してください。

また、 Random::DEFAULT は Random クラスオブジェクトが返ります。 


Random::DEFAULT == Random # => true
Random.rand(10) # => 4
