instance method Logger#error?

error? -> bool[permalink][rdoc]

現在の Logger オブジェクトが ERROR 以上のログレベルのメッセージを記録するなら真を返します。 

例

require 'logger'

logger = Logger.new(STDOUT)
logger.error? # => true
logger.level = Logger::Severity::FATAL
logger.error? # => false
