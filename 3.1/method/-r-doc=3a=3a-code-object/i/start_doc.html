instance method RDoc::CodeObject#start_doc

start_doc -> ()[permalink][rdoc]

以降に解析したコメントを RDoc::CodeObject#stop_doc を呼び出すまでの間、ドキュメントに含めます。

:startdoc: を見つけた時に呼び出されます。 RDoc::CodeObject#document_self と RDoc::CodeObject#document_children を true に設定します。
