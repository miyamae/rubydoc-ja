instance method Float#infinite?

infinite? -> 1 | -1 | nil[permalink][rdoc]

数値が +∞ のとき 1、-∞のとき -1 を返します。それ以外は nil を返します。 

例

inf = 1.0/0
p inf            # => Infinity
p inf.infinite?  # => 1

inf = -1.0/0
p inf            # => -Infinity
p inf.infinite?  # => -1
