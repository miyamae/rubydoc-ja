singleton method OpenSSL::HMAC.hexdigest

hexdigest(digest, key, data) -> String[permalink][rdoc]

渡された digest と key を用いて data の HMAC を計算し、その値を16進文字列で返します。

digest には利用するハッシュ関数を表す文字列("md5", "sha256" など) を渡します。

[PARAM] digest:
利用するハッシュ関数
[PARAM] key:
利用する鍵の文字列
[PARAM] data:
HMAC を計算する文字列
