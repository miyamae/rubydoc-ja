instance method Pathname#chown

chown(owner, group) -> Integer[permalink][rdoc]

File.chown(owner, group, self.to_s) と同じです。

[PARAM] owner:
オーナーを指定します。
[PARAM] group:
グループを指定します。 
例

require 'pathname'

Pathname('testfile').stat.uid     # => 501
Pathname('testfile').chown(502, 12)
Pathname('testfile').stat.uid     # => 502
