instance method Regexp#casefold?

casefold? -> bool[permalink][rdoc]

正規表現が大文字小文字の判定をしないようにコンパイルされている時、真を返します。 

例

reg = Regexp.new("foobar", Regexp::IGNORECASE)
p reg.casefold? # => true

reg = Regexp.new("hogehoge")
p reg.casefold? # => false
