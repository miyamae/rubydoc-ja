instance method Rake::FileList#sub

sub(pattern, replace) -> Rake::FileList[permalink][rdoc]

自身に含まれるファイルリストのそれぞれのエントリに対して String#sub を実行し、結果を新しい Rake::FileList として返します。

例: 

FileList['a.c', 'b.c'].sub(/\.c$/, '.o')  => ['a.o', 'b.o']
