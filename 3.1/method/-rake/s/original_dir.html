singleton method Rake.original_dir

original_dir -> String[permalink][rdoc]

Rake アプリケーションを開始したディレクトリを返します。 


# Rakefile での記載例とする

task default: :test_rake_app
task :test_rake_app do
  Rake.original_dir # => "/path/to/dir"
end
