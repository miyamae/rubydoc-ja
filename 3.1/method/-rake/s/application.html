singleton method Rake.application

application -> Rake::Application[permalink][rdoc]

現在の Rake アプリケーションを返します。 


# Rakefile での記載例とする

require 'pp'

task default: :test_rake_app
task :test_rake_app do
  pp Rake.application
end
