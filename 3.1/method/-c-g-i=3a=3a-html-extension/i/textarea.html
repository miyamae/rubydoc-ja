instance method CGI::HtmlExtension#textarea

textarea(name = "", cols = 70, rows = 10) -> String[permalink][rdoc]

textarea 要素を生成します。

[PARAM] name:
name 属性の値を指定します。
[PARAM] cols:
cols 属性の値を指定します。
[PARAM] rows:
rows 属性の値を指定します。

例: 

textarea("name")
  # = textarea({ "NAME" => "name", "COLS" => 70, "ROWS" => 10 })
textarea(attributes) -> String[permalink][rdoc]

textarea 要素を生成します。

[PARAM] attributes:
属性をハッシュで指定します。

例: 

textarea("name", 40, 5)
  # = textarea({ "NAME" => "name", "COLS" => 40, "ROWS" => 5 })
