instance method Kernel#find_type

find_type(type, opt, *headers) -> Array[permalink][rdoc] [added by mkmf]
find_type(type, opt, *headers) { ... } -> Array [added by mkmf]

静的な型 type がシステムに存在するかどうか検査します。

[PARAM] type:
検査したい型の名前を指定します。
[PARAM] opt:
コンパイラに渡す追加のオプションを指定します。
[PARAM] headers:
追加のヘッダを指定します。
