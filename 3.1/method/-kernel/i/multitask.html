instance method Kernel#multitask

multitask(args) { ... } -> Rake::MultiTask[permalink][rdoc] [added by rake]

事前タスクを並列実行するタスクを定義します。

与えられた事前タスクを実行する順序は不定です。

例: 

multitask :deploy => [:deploy_gem, :deploy_rdoc]
