instance method Rake::Application#init

init(app_name = 'rake')[permalink][rdoc]

コマンドラインオプションとアプリケーション名を初期化します。 

例

# Rakefile での記載例とする

task default: :test
task :test

Rake.application.name # => "rake"
Rake.application.init("MyApp") # => ["default"]
Rake.application.name # => "MyApp"
