instance method Comparable#==

self == other -> bool[permalink][rdoc]

比較演算子 <=> をもとにオブジェクト同士を比較します。 <=> が 0 を返した時に、true を返します。それ以外を返した場合は、false を返します。

[PARAM] other:
自身と比較したいオブジェクトを指定します。 
例

1 == 1   # => true
1 == 2   # => false
