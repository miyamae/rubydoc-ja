instance method StringScanner#string=

string=(str)[permalink][rdoc]

スキャン対象の文字列を str に変更して、マッチ記録を捨てます。

[PARAM] str:
スキャン対象の文字列を str に変更して、マッチ記録を捨てます。
[RETURN]
str を返します。 
例

require 'strscan'

str = '0123'
s = StringScanner.new('test string')
s.string = str     # => "0123"
s.scan(/\w+/)      # => "0123"
