instance method StringScanner#pointer

pointer -> Integer[permalink][rdoc]
pos -> Integer

現在のスキャンポインタのインデックスを返します。 

例

require 'strscan'

s = StringScanner.new('test string')
s.pos         # => 0
s.scan(/\w+/) # => "test"
s.pos         # => 4
s.scan(/\w+/) # => nil
s.pos         # => 4
s.scan(/\s+/) # => " "
s.pos         # => 5
