singleton method StringScanner.new

new(str, dup = false) -> StringScanner[permalink][rdoc]

新しい StringScanner オブジェクトを生成します。

[PARAM] str:
スキャン対象の文字列を指定します。
[PARAM] dup:
dup は単に無視します。引数の文字列は複製も freeze もされず、そのまま使います。 
例

require 'strscan'

s = StringScanner.new('This is an example string')
s.eos?            #=> false

p s.scan(/\w+/)   #=> "This"
p s.scan(/\w+/)   #=> nil
p s.scan(/\s+/)   #=> " "
