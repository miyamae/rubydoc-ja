instance method Symbol#encoding

encoding -> Encoding[permalink][rdoc]

シンボルに対応する文字列のエンコーディング情報を表現した Encoding オブジェクトを返します。

例: 

# encoding: utf-8

:foo.encoding        # => #<Encoding:US-ASCII>
:あかさたな.encoding   # => #<Encoding:UTF-8>
