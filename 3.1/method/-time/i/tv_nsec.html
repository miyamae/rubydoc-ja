instance method Time#nsec

nsec -> Integer[permalink][rdoc]
tv_nsec -> Integer

時刻のナノ秒の部分を整数で返します。 


t = Time.local(2000,1,2,3,4,5,6)
p "%10.9f" % t.to_f   # => "946749845.000005960"
p t.nsec              # => 6000

IEEE 754 浮動小数点数で表現できる精度が違うため、Time#to_fの最小の桁とnsecの最小の桁は異なります。nsecで表される値の方が正確です。
