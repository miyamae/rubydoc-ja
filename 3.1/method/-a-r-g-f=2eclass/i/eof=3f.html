instance method ARGF.class#eof

eof -> bool[permalink][rdoc]
eof? -> bool

現在開いているファイルがEOFに達したらtrueを返します。そうでない場合は falseを返します。

[EXCEPTION] IOError:
ファイルがopenされていない場合に発生します。 
$ echo "eof" | ruby argf.rb

ARGF.eof?                 # => false
3.times { ARGF.readchar }
ARGF.eof?                 # => false
ARGF.readchar             # => "\n"
ARGF.eof?                 # => true
