library rubygems/digest/digest_adapter

要約

Ruby 1.8.5 と 1.8.6 の間に存在する API の差異を吸収するためのライブラリです。

古いバージョンの API を新しいバージョンのものに合わせます。 RubyGems では digest, hexdigest の二つのメソッドが使用されるのでこの二つのメソッドのみ API を変換するようにしています。

Ruby 1.8.6 以降ではこのライブラリは使用されません。
