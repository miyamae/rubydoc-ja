特異メソッド

each {|entry| ... } -> Etc::Group[permalink][rdoc]
each -> Enumerator

/etc/group に含まれるエントリを一つずつブロックに渡して評価します。ブロックを省略した場合は Enumerator を返します。

[SEE_ALSO] Etc.#getpwent
