要約

false のクラス。 false は FalseClass クラスの唯一のインスタンスです。 false は nil オブジェクトとともに偽を表し、その他の全てのオブジェクトは真です。

self & other -> false[permalink][rdoc]

常に false を返します。

[PARAM] other:
論理積を行なう式です。

& は再定義可能な演算子に分類されていますので、通常は false & other の形で使われます。 

例

p false & true     #=> false
p false & false    #=> false
p false & nil      #=> false
p false & (1 == 1) #=> false
p false & (1 + 1)  #=> false

p false.&(true)    #=> false
p false.&(false)   #=> false
p false.&(nil)     #=> false
p false.&(1 == 1)  #=> false
p false.&(1 + 1)   #=> false
