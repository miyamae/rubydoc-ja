module JSON::Generator::GeneratorMethods::FalseClass

クラス・モジュールの継承リスト:
  1. JSON::Generator::GeneratorMethods::FalseClass

aliases: JSON::Ext::Generator::GeneratorMethods::FalseClass

要約

FalseClass に JSON で使用するインスタンスメソッドを追加するためのモジュールです。

目次

インスタンスメソッド
  • to_json

インスタンスメソッド

to_json(state_or_hash = nil) -> String[permalink][rdoc]

自身から生成した JSON 形式の文字列を返します。

"false" という文字列を返します。

[PARAM] state_or_hash:
生成する JSON 形式の文字列をカスタマイズするために JSON::State のインスタンスか、 JSON::State.new の引数と同じ Hash を指定します。 
例

require "json"

false.to_json # => "false"
