class Rinda::TupleSpaceProxy

クラス・モジュールの継承リスト:
  1. BasicObject
  2. Kernel
  3. Object
  4. Rinda::TupleSpaceProxy

要約

リモートの Rinda::TupleSpace オブジェクトを包むプロクシクラスです。

Rinda::TupleSpace#take でタプルの受け渡し時にタプルが消失する可能性を下げるためのプロクシクラスです。
