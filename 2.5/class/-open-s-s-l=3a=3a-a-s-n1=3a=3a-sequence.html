特異メソッド

new(value) -> OpenSSL::ASN1::Sequence[permalink][rdoc]
new(value, tag, tagging, tag_class) -> OpenSSL::ASN1::Sequence

ASN.1 の Sequence 型の値を表現する OpenSSL::ASN1::Sequence オブジェクトを生成します。

value 以外の引数を省略した場合はタグクラスは :UNIVERSAL、タグ は OpenSSL::ASN1::SEQUENCE となります。

[PARAM] value:
ASN.1値を表すRubyのオブジェクト(OpenSSL::ASN1::ASN1Dataの配列)
[PARAM] tag:
タグ番号
[PARAM] tagging:
タグ付けの方法(:IMPLICIT もしくは :EXPLICIT)
[PARAM] tag_class:
タグクラス(:UNIVERSAL, :CONTEXT_SPECIFIC, :APPLICATION, :PRIVATE のいずれか)
