new(value) -> OpenSSL::ASN1::UTCTime[permalink][rdoc]
new(value, tag, tagging, tag_class) -> OpenSSL::ASN1::UTCTime

ASN.1 の UTCTime 型の値を表現する OpenSSL::ASN1::UTCTime オブジェクトを生成します。

value 以外の引数を省略した場合はタグクラスは :UNIVERSAL、タグ は OpenSSL::ASN1::UTCTIME となります。

[PARAM] value:
ASN.1 値を表す Ruby のオブジェクト(Timeオブジェクト)
[PARAM] tag:
タグ番号
[PARAM] tagging:
タグ付けの方法(:IMPLICIT もしくは :EXPLICIT)
[PARAM] tag_class:
タグクラス(:UNIVERSAL, :CONTEXT_SPECIFIC, :APPLICATION, :PRIVATE のいずれか)
