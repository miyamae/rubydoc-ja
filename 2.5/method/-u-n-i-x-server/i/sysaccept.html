instance method UNIXServer#sysaccept

sysaccept -> Integer[permalink][rdoc]

接続したクライアントのソケットをファイル記述子で返すことを除けば UNIXServer#accept と同じです。

例: 

require 'socket'

UNIXServer.open("/tmp/s") {|serv|
  c = UNIXSocket.open("/tmp/s")
  p serv.sysaccept        #=> 6
}
