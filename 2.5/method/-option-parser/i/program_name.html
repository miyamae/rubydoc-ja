instance method OptionParser#program_name

program_name -> String[permalink][rdoc]

プログラムの名前を文字列で返します。

デフォルトは $0 が使われます。

[RETURN]
プログラムの名前を文字列で返します。 
例

require "optparse"

OptionParser.new do |opts|
  p $0                 # => /path/to/filename.rb
  p opts.program_name  # => filename
end
