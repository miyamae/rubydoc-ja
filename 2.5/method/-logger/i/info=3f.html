instance method Logger#info?

info? -> bool[permalink][rdoc]

現在の Logger オブジェクトが INFO 以上のログレベルのメッセージを記録するなら真を返します。 

例

require 'logger'

logger = Logger.new(STDOUT)
logger.info? # => true
logger.level = Logger::Severity::ERROR
logger.info? # => false
