instance method Gem::Commands::WhichCommand#find_paths

find_paths(package_name, dirs) -> Array[permalink][rdoc]

dirs から package_name という名前を持つファイルを探索します。

以下の拡張子を持つファイルが対象です。 

%w[.rb .rbw .so .dll .bundle]
[PARAM] package_name:
ファイルの名前を指定します。
[PARAM] dirs:
探索するディレクトリを文字列の配列で指定します。
