instance method CGI::QueryExtension#multipart?

multipart? -> bool[permalink][rdoc]

マルチパートフォームの場合は、真を返します。そうでない場合は、偽を返します。 

例：
cgi = CGI.new
if cgi.multipart?
  field1=cgi['field1'].read
else
  field1=cgi['field1']
end
