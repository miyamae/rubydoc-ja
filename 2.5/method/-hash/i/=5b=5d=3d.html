instance method Hash#[]=

self[key] = value[permalink][rdoc]
store(key, value) -> object

key に対して value を関連づけます。value を返します。

[PARAM] key:
キーを指定します。
[PARAM] value:
値を指定します。 
例

h = {}

h[:key] = "value"
p h #=>{:key => "value"}
