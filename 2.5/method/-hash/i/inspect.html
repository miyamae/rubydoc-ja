instance method Hash#inspect

to_s -> String[permalink][rdoc]
inspect -> String

ハッシュの内容を人間に読みやすい文字列にして返します。 

例

h = { "c" => 300, "a" => 100, "d" => 400  }
h.inspect   # => "{\"c\"=>300, \"a\"=>100, \"d\"=>400}"
