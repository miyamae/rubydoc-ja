instance method Enumerable#include?

member?(val) -> bool[permalink][rdoc]
include?(val) -> bool

val と == の関係にある要素を含むとき真を返します。

[PARAM] val:
任意のオブジェクト 
例

[2, 4, 6].include? 2 #=> true
[2, 4, 6].include? 1 #=> false
[2, 4, 6].member? 2  #=> true
[2, 4, 6].member? 1  #=> false
