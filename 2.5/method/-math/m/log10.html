module function Math.#log10

log10(x) -> Float[permalink][rdoc]

x の常用対数を返します。

[PARAM] x:
正の実数
[EXCEPTION] TypeError:
xに数値以外を指定した場合に発生します。
[EXCEPTION] Math::DomainError:
x に範囲外の実数を指定した場合に発生します。
[EXCEPTION] RangeError:
xに実数以外の数値を指定した場合に発生します。 
例

Math.log10(1)       # => 0.0
Math.log10(10)      # => 1.0
Math.log10(10**100) # => 100.0
