instance method StringScanner#charpos

charpos -> Integer[permalink][rdoc]

現在のスキャンポインタのインデックスを文字単位で返します。 

例

require 'strscan'

s = StringScanner.new("abcädeföghi")
s.charpos           # => 0
s.scan_until(/ä/)   # => "abcä"
s.pos               # => 5
s.charpos           # => 4
