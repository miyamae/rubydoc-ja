instance method Time#saturday?

saturday? -> bool[permalink][rdoc]

self の表す時刻が土曜日である場合に true を返します。そうでない場合に false を返します。 


t = Time.local(2006, 6, 10)      # => 2006-06-10 00:00:00 +0900
p t.saturday?                    # => true
