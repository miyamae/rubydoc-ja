instance method File#chmod

chmod(mode) -> 0[permalink][rdoc]

ファイルのモードを指定された mode に変更します。

モードの変更に成功した場合は 0 を返します。失敗した場合は例外 Errno::EXXX が発生します。

[PARAM] mode:
chmod(2) と同様に整数で指定します。
[EXCEPTION] IOError:
自身が close されている場合に発生します。
[EXCEPTION] Errno::EXXX:
失敗した場合に発生します。 
例

f = File.new("out", "w");
f.chmod(0644)   #=> 0
