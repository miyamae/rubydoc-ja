instance method Pathname#delete

unlink -> Integer[permalink][rdoc]
delete -> Integer

self が指すディレクトリあるいはファイルを削除します。 

例

require "pathname"

pathname = Pathname("/path/to/sample")
pathname.exist? # => true
pathname.unlink # => 1
pathname.exist? # => false
