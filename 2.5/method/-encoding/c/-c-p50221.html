constant Encoding::CP50221

CP50221 -> Encoding[permalink][rdoc]

Windows で用いられる、ISO-2022-JP 亜種です。

ISO-2022-JP に加え、ESC ( I でいわゆる半角カナを許し、Windows の機種依存文字を扱うことができます。
