instance method Thread#stop?

stop? -> bool[permalink][rdoc]

スレッドが終了(dead)あるいは停止(stop)している時、true を返します。 

例

a = Thread.new { Thread.stop }
b = Thread.current
a.stop?   # => true
b.stop?   # => false
