instance method Thread#add_trace_func

add_trace_func(pr) -> Proc[permalink][rdoc]

スレッドにトレース用ハンドラを追加します。

追加したハンドラを返します。

[PARAM] pr:
トレースハンドラ(Proc オブジェクト) 
例

th = Thread.new do
  class Trace
  end
  43.to_s
end
th.add_trace_func lambda {|*arg| p arg }
th.join
