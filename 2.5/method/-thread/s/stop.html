singleton method Thread.stop

stop -> nil[permalink][rdoc]

他のスレッドから Thread#run メソッドで再起動されるまで、カレントスレッドの実行を停止します。 

例

a = Thread.new { print "a"; Thread.stop; print "c" }
sleep 0.1 while a.status!='sleep'
print "b"
a.run
a.join
# => "abc"
