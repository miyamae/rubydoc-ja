instance method Array#drop

drop(n) -> Array[permalink][rdoc]

配列の先頭の n 要素を捨てて、残りの要素を配列として返します。このメソッドは自身を破壊的に変更しません。

[PARAM] n:
捨てる要素数。 
例

a = [1, 2, 3, 4, 5, 0]
a.drop(3)             # => [4, 5, 0]

# 変数aの値は変化しない
a                     # => [1, 2, 3, 4, 5, 0]
