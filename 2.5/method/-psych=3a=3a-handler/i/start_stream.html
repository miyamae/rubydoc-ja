instance method Psych::Handler#start_stream

start_stream(encoding) -> ()[permalink][rdoc]

YAML ストリームの始まりで呼び出されます。

encoding にはストリームのエンコーディング(以下のいずれか)が渡されます。

  • Psych::Parser::UTF8
  • Psych::Parser::UTF16BE
  • Psych::Parser::UTF16LE

このメソッドは YAML のストリームごとに呼び出されます。一つのストリームには複数のドキュメントが含まれている可能性があります。

必要に応じてこのメソッドを override してください。

[PARAM] encoding:
ストリームのエンコーディング(整数値)
