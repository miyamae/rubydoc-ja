instance method Kernel#j

j(*objects) -> nil[permalink][rdoc] [added by json]

与えられたオブジェクトを JSON 形式の文字列で標準出力に一行で出力します。

[PARAM] objects:
JSON 形式で出力したいオブジェクトを指定します。 
例

require "json"

j([1,2,{"name" => "tanaka","age" => 19}])
# => [1,2,{"name":"tanaka","age":19}]
