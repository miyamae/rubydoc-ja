instance method Kernel#rm

rm -> ()[permalink][rdoc] [added by un]

ファイルやディレクトリを削除します。 

ruby -run -e rm -- [OPTION] FILE

-f          存在しないファイルを無視します
-r          ディレクトリを再帰的にたどってファイルやディレクトリを削除します
-v          詳細表示
