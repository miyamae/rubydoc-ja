instance method Integer#%

self % other -> Numeric[permalink][rdoc]
modulo(other) -> Numeric

算術演算子。剰余を計算します。 


13 % 4    # =>  1
13 % -4   # => -3
-13 % 4   # =>  3
-13 % -4  # => -1
[PARAM] other:
二項演算の右側の引数(対象)
[RETURN]
計算結果
