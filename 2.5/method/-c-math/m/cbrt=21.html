module function CMath.#cbrt!

cbrt!(x) -> Float[permalink][rdoc]

実数 x の立方根を返します。Math.#cbrt のエイリアスです。

[PARAM] x:
実数
[EXCEPTION] TypeError:
x に数値以外を指定した場合に発生します。
[EXCEPTION] RangeError:
x に実数以外の数値を指定した場合に発生します。 
例

require "cmath"
CMath.cbrt!(8.0)# => 2.0
CMath.cbrt!(-8.0) # => -2.0
