module function CMath.#log10!

log10!(x) -> Float[permalink][rdoc]

実数 x の常用対数を返します。Math.#log10 のエイリアスです。

[PARAM] x:
真数を正の実数で指定します。
[EXCEPTION] Math::DomainError:
x が負の数である場合に発生します。
[EXCEPTION] TypeError:
x に数値以外を指定した場合に発生します。
[EXCEPTION] RangeError:
x に実数以外の数値を指定した場合に発生します。
