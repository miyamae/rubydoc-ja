instance method Gem::Version#marshal_load

marshal_load(array) -> nil[permalink][rdoc]

ダンプされた情報をロードし、自身を破壊的に変更します。 


version = Gem::Version.new('')
version.marshal_load(["1.2.0a"])
p version # => #<Gem::Version "1.2.0a">
[PARAM] array:
バージョン情報を含む配列を指定します。
