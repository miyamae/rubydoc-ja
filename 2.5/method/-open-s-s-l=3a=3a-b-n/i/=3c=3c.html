instance method OpenSSL::BN#<<

self << other -> OpenSSL::BN[permalink][rdoc]

自身を other ビット左シフトした値を返します。 


bn = 1.to_bn
pp bn << 1    # => #<OpenSSL::BN 2>
pp bn         # => #<OpenSSL::BN 1>
[PARAM] other:
シフトするビット数
[EXCEPTION] OpenSSL::BNError:
計算時エラー
