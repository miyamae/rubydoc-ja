instance method CGI::HtmlExtension#base

base(href = "") -> String[permalink][rdoc]

base 要素を生成します。

[PARAM] href:
文字列を指定します。属性をハッシュで指定することもできます。

例: 

base("http://www.example.com/cgi")
  # => "<BASE HREF=\"http://www.example.com/cgi\">"
