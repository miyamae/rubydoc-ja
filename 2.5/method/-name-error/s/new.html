singleton method NameError.new

new(error_message = "", name = nil) -> NameError[permalink][rdoc]

例外オブジェクトを生成して返します。

[PARAM] error_message:
エラーメッセージを表す文字列です
[PARAM] name:
未定義だったシンボルです

例: 

err = NameError.new("message", "foo")
p err       # => #<NameError: message>
p err.name  # => "foo"
