instance method Shell::CommandProcessor#open

open(path, mode) -> File | Dir[permalink][rdoc]

path がファイルなら、 File.open path がディレクトリなら、 Dir.open の動作をします。

[PARAM] path:
開きたいパスを指定します。
[PARAM] mode:
アクセスモードを指定します。path がディレクトリの場合は無視されます。

[SEE_ALSO] File.open, Dir.open
