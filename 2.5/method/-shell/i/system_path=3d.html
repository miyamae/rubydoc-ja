instance method Shell#system_path

system_path -> Array[permalink][rdoc]
system_path=(path)

コマンドサーチパスの配列を返す。

[PARAM] path:
コマンドサーチパスの配列を指定します。

使用例 

require 'shell'
sh = Shell.new
sh.system_path = [ "./" ]
p sh.system_path #=> ["./"]
