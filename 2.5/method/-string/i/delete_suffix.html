instance method String#delete_suffix

delete_suffix(suffix) -> String[permalink][rdoc]

文字列の末尾から suffix を削除した文字列のコピーを返します。

[PARAM] suffix:
末尾から削除する文字列を指定します。
[RETURN]
文字列の末尾から suffix を削除した文字列のコピー 

"hello".delete_suffix("llo") # => "he"
"hello".delete_suffix("hel") # => "hello"
