instance method File#size

size -> Integer[permalink][rdoc]

ファイルのサイズを返します。

例: 

File.open("/dev/null") do |f|
  f.size   #=> 0
end
[EXCEPTION] IOError:
自身が close されている場合に発生します。
[EXCEPTION] Errno::EXXX:
失敗した場合に発生します。

[SEE_ALSO] File#lstat
