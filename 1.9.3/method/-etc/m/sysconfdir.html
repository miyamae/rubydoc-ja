module function Etc.#sysconfdir

sysconfdir -> String | nil[permalink][rdoc]

システムの設定ディレクトリを返します。 

require 'etc'
p Etc.sysconfdir # => "/etc"

主に "/etc" を返しますが、Ruby をソースからビルドした場合は異なるディレ クトリを返す場合があります。例えば、Ruby を /usr/local にインストールし た場合は、"/usr/local/etc" を返します。
