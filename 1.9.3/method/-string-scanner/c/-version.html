constant StringScanner::Version

Version -> String[permalink][rdoc]

StringScanner クラスのバージョンを文字列で返します。 この文字列は Object#freeze されています。 

StringScanner::Version           # => "0.7.0"
StringScanner::Version.frozen?   # => true
