instance method Hash#has_key?

has_key?(key) -> bool[permalink][rdoc]
include?(key) -> bool
key?(key) -> bool
member?(key) -> bool

ハッシュが key をキーとして持つ時真を返します。

[PARAM] key:
探索するキーを指定します。 
p({1 => "one"}.has_key?(1)) #=> true
p({1 => "one"}.has_key?(2)) #=> false
