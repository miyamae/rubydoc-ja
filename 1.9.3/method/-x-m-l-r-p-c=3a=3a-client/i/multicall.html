instance method XMLRPC::Client#multicall

multicall(*methods)[permalink][rdoc]

You can use this method to execute several methods on a XMLRPC server which supports the multi-call extension. Example: 

s.multicall(
  ['michael.add', 3, 4],
  ['michael.sub', 4, 5]
)
# => [7, -1]
