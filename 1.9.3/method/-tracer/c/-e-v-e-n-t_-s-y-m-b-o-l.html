constant Tracer::EVENT_SYMBOL

EVENT_SYMBOL[permalink][rdoc]

トレース出力のシンボルのハッシュです。 下記のような文字列があります。 

EVENT_SYMBOL = {
  "line" => "-",
  "call" => ">",
  "return" => "<",
  "class" => "C",
  "end" => "E",
  "c-call" => ">",
  "c-return" => "<",
}
