instance method Rake::GemPackageTask#gem_spec

gem_spec -> Gem::Specification[permalink][rdoc]

package ターゲットで使用する gemspec を返します。

gemspec にはパッケージ名、バージョン、パッケージに含まれるファイルなどが定義 されているので、それらを明示的に指定する必要はありません。
