singleton method Dir.exist?

exist?(file_name) -> bool[permalink][rdoc]
exists?(file_name) -> bool

file_name で与えられたディレクトリが存在する場合に真を返します。 そうでない場合は、偽を返します。

[PARAM] file_name:
存在を確認したいディレクトリ名。 
Dir.exist?(".")      # => true
Dir.exists?(".")     # => true
File.directory?(".") # => true

[SEE_ALSO] File.directory?
