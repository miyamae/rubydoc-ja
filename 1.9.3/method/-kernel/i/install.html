instance method Kernel#install

install -> ()[permalink][rdoc] [added by un]

ファイルをコピーし、その属性を設定します。 

ruby -run -e install -- [OPTION] SOURCE DEST

-p          ファイルのアクセス時刻と修正時刻を保持します。
-m          chmod と同じようにファイルのパーミッションを設定します。
-v          詳細表示
