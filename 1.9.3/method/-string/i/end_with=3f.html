instance method String#end_with?

end_with?(*strs) -> bool[permalink][rdoc]

self の末尾が strs のいずれかであるとき true を返します。

[PARAM] strs:
パターンを表す文字列 (のリスト)

例: 

"string".end_with?("ing")          # => true
"string".end_with?("str")          # => false
"string".end_with?("str", "ing")   # => true
