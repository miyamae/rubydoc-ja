instance method DBM#select

select {|key, value| ... } -> [Array][permalink][rdoc]

ブロックを評価して真になった要素のみを配列に格納して返します。 

require 'dbm'

db1 = DBM.open('aaa.db', 0666, DBM::NEWDB)
db1[:a] = 'aaa'
db1[:b] = 'bbbbbb'

p db1.select {|key, value| key == 'a' } #=> [["a", "aaa"]]
