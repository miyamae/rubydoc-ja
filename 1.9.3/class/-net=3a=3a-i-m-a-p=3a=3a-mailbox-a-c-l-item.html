class Net::IMAP::MailboxACLItem

クラスの継承リスト: Net::IMAP::MailboxACLItem < Struct < Enumerable < Object < Kernel < BasicObject

要約

GETACL の応答の各要素を表すクラスです。

Net::IMAP#getacl の返り値として用いられます。

詳しくは [RFC2086] を参照してください。

インスタンスメソッド

rights -> String[permalink][rdoc]

アクセス権限を文字列で返します。

Net::IMAP::MailboxACLItem#user で得られるユーザが 持っている権限が返されます。

この文字列の意味については [RFC2086] を参照してください。

user -> String[permalink][rdoc]

ユーザ名を返します。

このユーザは Net::IMAP#getacl で指定したメールボックスに 対し何らかの権限を持っています。
