class IRB::Frame

クラスの継承リスト: IRB::Frame < Object < Kernel < BasicObject

要約

現在実行中のフレーム情報を取り扱うためのクラスです。

[注] set_trace_func を用いて Ruby の実行をトレースしています。 マルチスレッドには対応していません。
