class Gem::ConfigFile

クラスの継承リスト: Gem::ConfigFile < Object < Kernel < BasicObject

要約

設定ファイルに書かれている gem コマンドのオプションを オブジェクトに保存するためのクラスです。

このクラスのインスタンスはハッシュのように振る舞います。
