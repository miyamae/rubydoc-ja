class Resolv::DNS::Resource::MX

クラスの継承リスト: Resolv::DNS::Resource::MX < Resolv::DNS::Resource < Resolv::DNS::Query < Object < Kernel < BasicObject

要約

DNS リソースの MX レコード を表す抽象クラスです。

Resolv::DNS#getresource で MX レコードを得たい場合は Resolv::DNS::Resource::IN::MX を使うべきです。
