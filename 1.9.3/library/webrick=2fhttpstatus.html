library webrick/httpstatus

要約

HTTP のステータスを表す例外クラスを提供します。

モジュール

WEBrick::HTTPStatus

HTTP のステータスを表す例外クラスを提供するモジュールです。 ステータスコード 200 などの成功の場合も含まれます。
