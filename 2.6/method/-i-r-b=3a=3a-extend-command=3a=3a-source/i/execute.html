instance method IRB::ExtendCommand::Source#execute

execute(file_name) -> nil[permalink][rdoc]

現在の irb インタプリタ上で、 Ruby スクリプト path を評価します。

path の内容を irb で一行ずつタイプしたかのように、irb 上で一行ずつ評価されます。$" は更新されず、何度でも実行し直す事ができます。

[PARAM] file_name:
ファイル名を文字列で指定します。
