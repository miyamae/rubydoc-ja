singleton method ENV.slice

slice(*keys) -> Hash[permalink][rdoc]

引数で指定されたキーとその値だけを含む Hash を返します。 

例

ENV["foo"] = "bar"
ENV["baz"] = "qux"
ENV["bar"] = "rab"
ENV.slice()             # => {}
ENV.slice("")           # => {}
ENV.slice("unknown")    # => {}
ENV.slice("foo", "baz") # => {"foo"=>"bar", "baz"=>"qux"}
