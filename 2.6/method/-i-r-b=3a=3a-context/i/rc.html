instance method IRB::Context#rc

rc -> bool[permalink][rdoc]
rc? -> bool

~/.irbrc などの設定ファイルがあれば読み込みを行うかどうかを返します。

[RETURN]
設定ファイルの読み込みを行う場合は true を返します。行わない場合(irb の起動時に -f オプションを指定した場合)は false を返します。
