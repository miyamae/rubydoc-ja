instance method Kernel#DelegateClass

DelegateClass(superclass) -> object[permalink][rdoc] [added by delegate]

クラス superclass のインスタンスへメソッドを委譲するクラスを定義し、そのクラスを返します。

[PARAM] superclass:
委譲先となるクラス

例: 

require 'delegate'

class ExtArray < DelegateClass(Array)
  def initialize
    super([])
  end
end
a = ExtArray.new
p a.class   # => ExtArray
a.push 25
p a         # => [25]
