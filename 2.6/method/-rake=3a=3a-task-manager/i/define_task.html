instance method Rake::TaskManager#define_task

define_task(task_class, *args) { ... } -> Rake::Task[permalink][rdoc]

タスクを定義します。

[PARAM] task_class:
タスククラスを指定します。
[PARAM] args:
タスクに渡すパラメータを指定します。 

# Rakefile での記載例とする

task default: :test_rake_app
task :test_rake_app do
  Rake.application.define_task(Rake::Task, :t) # => <Rake::Task t => []>
end
