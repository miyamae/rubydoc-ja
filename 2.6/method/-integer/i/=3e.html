instance method Integer#>

self > other -> bool[permalink][rdoc]

比較演算子。数値として大きいか判定します。

[PARAM] other:
比較対象の数値
[RETURN]
self よりも other の方が小さい場合 true を返します。そうでなければ false を返します。 

1 > 0    # => true
1 > 1    # => false
