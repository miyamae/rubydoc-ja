instance method Integer#**

self ** other -> Numeric[permalink][rdoc]
pow(other) -> Numeric
pow(other, modulo) -> Integer

算術演算子。冪(べき乗)を計算します。

[PARAM] other:
二項演算の右側の引数(対象)
[PARAM] modulo:
指定すると、計算途中に巨大な値を生成せずに (self**other) % modulo と同じ結果を返します。
[RETURN]
計算結果
[EXCEPTION] TypeError:
2引数 pow で Integer 以外を指定した場合に発生します。
[EXCEPTION] RangeError:
2引数 pow で other に負の数を指定した場合に発生します。 

2 ** 3 # => 8
2 ** 0 # => 1
0 ** 0 # => 1
3.pow(3,  8)  # =>  3
3.pow(3, -8)  # => -5
3.pow(2, -2)  # => -1
-3.pow(3,  8) # =>  5
-3.pow(3, -8) # => -3
5.pow(2, -8)  # => -7
