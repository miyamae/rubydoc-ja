instance method ARGF.class#binmode?

binmode? -> bool[permalink][rdoc]

ARGF の入力ストリームがバイナリモードなら true を返します。そうでない場合、false を返します。

バイナリモードにするためには ARGF.class#binmode を使用します。 

ARGF.binmode? # => false
ARGF.binmode
ARGF.binmode? # => true
