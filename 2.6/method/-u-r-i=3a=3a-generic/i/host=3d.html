instance method URI::Generic#host=

host=(s)[permalink][rdoc]

自身の host を設定します。

[PARAM] s:
自身の host を表す文字列を指定します。
[EXCEPTION] URI::InvalidComponentError:
不正な引数 s に対して発生します。
[EXCEPTION] URI::InvalidURIError:
host と registry を同時に設定した場合に発生します。
