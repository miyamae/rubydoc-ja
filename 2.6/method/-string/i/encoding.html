instance method String#encoding

encoding -> Encoding[permalink][rdoc]

文字列のエンコーディング情報を表現した Encoding オブジェクトを返します。 

例

# encoding: utf-8
utf8_str = "test"
euc_str = utf8_str.encode("EUC-JP")
utf8_str.encoding   # => #<Encoding:UTF-8>
euc_str.encoding    # => #<Encoding:EUC-JP>
