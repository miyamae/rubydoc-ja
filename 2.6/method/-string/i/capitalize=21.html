instance method String#capitalize!

capitalize!(*options) -> self | nil[permalink][rdoc]

文字列先頭の文字を大文字に、残りを小文字に破壊的に変更します。

[PARAM] options:
オプションの詳細は String#downcase を参照してください。
[RETURN]
capitalize! は self を変更して返しますが、変更が起こらなかった場合は nil を返します。 
例

str = "foobar"
str.capitalize!
p str   # => "Foobar"

str = "fooBAR"
str.capitalize!
p str   # => "Foobar"
