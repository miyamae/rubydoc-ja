instance method String#ascii_only?

ascii_only? -> bool[permalink][rdoc]

文字列がASCII文字のみで構成されている場合に true を返します。さもなくば false を返します。

例: 

'abc123'.ascii_only?        # => true
''.ascii_only?              # => true
'日本語'.ascii_only?        # => false
'日本語abc123'.ascii_only?  # => false
