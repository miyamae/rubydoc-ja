instance method String#delete_prefix!

delete_prefix!(prefix) -> self | nil[permalink][rdoc]

self の先頭から破壊的に prefix を削除します。

[PARAM] prefix:
先頭から削除する文字列を指定します。
[RETURN]
削除した場合は self、変化しなかった場合は nil 

"hello".delete_prefix!("hel") # => "lo"
"hello".delete_prefix!("llo") # => nil
