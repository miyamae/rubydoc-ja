instance method Array#collect

collect -> Enumerator[permalink][rdoc]
map -> Enumerator
collect {|item| ... } -> [object]
map {|item| ... } -> [object]

各要素に対してブロックを評価した結果を全て含む配列を返します。

ブロックを省略した場合は Enumerator を返します。 

例

# すべて 3 倍にする
p [1, 2, 3].map {|n| n * 3 }  # => [3, 6, 9]

[SEE_ALSO] Hash#to_h, Enumerable#collect, Enumerable#map
