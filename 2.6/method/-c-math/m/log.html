module function CMath.#log

log(z) -> Float | Complex[permalink][rdoc]
log(z, b) -> Float | Complex

z の対数を返します。

[PARAM] z:
真数を指定します。
[PARAM] b:
底を指定します。省略した場合は自然対数を計算します。
[EXCEPTION] TypeError:
引数のどちらかに数値以外を指定した場合に発生します。 
例

require "cmath"
CMath.log(Complex(0, 0)) # => -Infinity+0.0i
CMath.log(0) # => -Infinity
