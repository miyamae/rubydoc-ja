instance method Time#monday?

monday? -> bool[permalink][rdoc]

self の表す時刻が月曜日である場合に true を返します。そうでない場合に false を返します。 


t = Time.local(2003, 8, 4)       # => 2003-08-04 00:00:00 +0900
p t.monday?                      # => true
