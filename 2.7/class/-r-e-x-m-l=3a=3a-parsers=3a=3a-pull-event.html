class REXML::Parsers::PullEvent

要約

REXML::Parsers::PullParser で使われるパース結果を表すイベントクラス。

REXML::Parsers::PullParser#pull および REXML::Parsers::PullParser#peek がこのクラスのオブジェクトを返します。
