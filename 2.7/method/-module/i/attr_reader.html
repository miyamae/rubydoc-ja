instance method Module#attr_reader

attr_reader(*name) -> nil[permalink][rdoc]

インスタンス変数 name の読み取りメソッドを定義します。

このメソッドで定義されるメソッドの定義は以下の通りです。 

例

def name
  @name
end
[PARAM] name:
String または Symbol を 1 つ以上指定します。
