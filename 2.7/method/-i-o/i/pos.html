instance method IO#pos

pos -> Integer[permalink][rdoc]
tell -> Integer

ファイルポインタの現在の位置を整数で返します。

[EXCEPTION] IOError:
既に close されている場合に発生します。 
例

IO.write("testfile", "This is line one\n")
File.open("testfile") do |f|
  f.pos    # => 0
  f.gets   # => "This is line one\n"
  f.pos    # => 17
end
