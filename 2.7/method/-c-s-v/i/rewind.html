instance method CSV#rewind

rewind -> 0[permalink][rdoc]

IO#rewind に似ています。CSV#lineno を 0 にします。 

例

require "csv"

csv = CSV.new("header1,header2\nrow1_1,row1_2")
csv.lineno # => 0
csv.readline
csv.lineno # => 1
csv.rewind
csv.lineno # => 0
