instance method Addrinfo#ip?

ip? -> bool[permalink][rdoc]

アドレスが IP (v4/v6) のものならば true を返します。 

require 'socket'

Addrinfo.tcp("127.0.0.1", 80).ip? #=> true
Addrinfo.tcp("::1", 80).ip?       #=> true
Addrinfo.unix("/tmp/sock").ip?    #=> false
