instance method Numeric#infinite?

infinite? -> nil[permalink][rdoc]

常に nil を返します。自身が Float かComplex、もしくはそのサブクラスのインスタンスの場合は、self の絶対値が負の無限大の場合に-1を、正の無限大の場合に1を、有限値の場合に nil を返します。 

例

10.infinite?     # => nil
(3r).infinite?   # => nil
