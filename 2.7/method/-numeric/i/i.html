instance method Numeric#i

i -> Complex[permalink][rdoc]

Complex(0, self) を返します。

ただし、Complex オブジェクトでは利用できません。 

例

10.i             # => (0+10i)
-10.i            # => (0-10i)
(0.1).i          # => (0+0.1i)
Rational(1, 2).i # => (0+(1/2)*i)
