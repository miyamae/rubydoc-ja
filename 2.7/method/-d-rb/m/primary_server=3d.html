module function DRb.#primary_server=

primary_server=(server)[permalink][rdoc]

ローカルプロセスのプライマリサーバーを変更します。

通常、プライマリサーバは DRb.#start_service などが設定するものであり、ユーザが変更すべきではありません。このメソッドは dRuby の内部構造を良く理解した上で利用してください。
