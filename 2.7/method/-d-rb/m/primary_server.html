module function DRb.#primary_server

primary_server -> DRb::DRbServer|nil[permalink][rdoc]

ローカルプロセスのプライマリサーバーを返します。

プライマリサーバとは DRb.#start_service によって起動されるサーバです。

サーバが動いていない場合は nil を返します。
