instance method Net::IMAP::ContentDisposition#param

param -> { String => String } | nil[permalink][rdoc]

Content-Disposition フィールドのパラメータをハッシュテーブルで返します。

ハッシュテーブルのキーは以下のような値を取ります。詳しくは [RFC2183] などを見てください。

  • "FILENAME"
  • "CREATION-DATE"
  • "MODIFICATION-DATE"
  • "READ-DAT"
  • "SIZE"
