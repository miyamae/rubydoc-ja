instance method String#upcase!

upcase!(*options) -> self | nil[permalink][rdoc]

全ての小文字を対応する大文字に破壊的に置き換えます。どの文字がどう置き換えられるかは、オプションの有無や文字列のエンコーディングに依存します。

[PARAM] options:
オプションの詳細は String#downcase を参照してください。 
例

buf = "stRIng? STring."
buf.upcase!
p buf   # => "STRING? STRING."
