singleton method Vector.basis

basis(size:, index:) -> Vector[permalink][rdoc]

size 次元ベクトル空間の index 番目の標準基底を返します。 

例

require 'matrix'
Vector.basis(size: 3, index: 1) # => Vector[0, 1, 0]
[PARAM] size:
ベクトルの次元
[PARAM] index:
標準基底の何番目か。0 origin
