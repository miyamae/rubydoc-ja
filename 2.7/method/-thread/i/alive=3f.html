instance method Thread#alive?

alive? -> bool[permalink][rdoc]

スレッドが「生きている」時、true を返します。

例: 

thr = Thread.new { }
thr.join                # => #<Thread:0x401b3fb0 dead>
Thread.current.alive?   # => true
thr.alive?              # => false

Thread#status が真を返すなら、このメソッドも真です。
