instance method Integer#floor

floor(ndigits = 0) -> Integer[permalink][rdoc]

self と等しいかより小さな整数のうち最大のものを返します。

[PARAM] ndigits:
10進数での小数点以下の有効桁数を整数で指定します。負の整数を指定した場合、小数点位置から左に少なくとも n 個の 0 が並びます。 

1.floor           # => 1
1.floor(2)        # => 1
18.floor(-1)      # => 10
(-18).floor(-1)   # => -20
