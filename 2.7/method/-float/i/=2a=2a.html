instance method Float#**

self ** other -> Float[permalink][rdoc]

算術演算子。冪を計算します。

[PARAM] other:
二項演算の右側の引数(対象) 
例

# 冪
1.2 ** 3.0  # => 1.7279999999999998
3.0 + 4.5 - 1.3 / 2.4 * 3 % 1.2 ** 3.0 # => 5.875
0.0 ** 0 # => 1.0
