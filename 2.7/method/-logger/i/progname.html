instance method Logger#progname

progname -> String[permalink][rdoc]

ログに出力するプログラム名を取得します。 

例

require 'logger'

logger = Logger.new(STDOUT)
logger.progname # => nil
logger.progname = "MyProgName"
logger.progname # => "MyProgName"
