instance method Array#include?

include?(val) -> bool[permalink][rdoc]

配列が val と == で等しい要素を持つ時に真を返します。

[PARAM] val:
オブジェクトを指定します。 
例

a = [ "a", "b", "c" ]
a.include?("b")       #=> true
a.include?("z")       #=> false
