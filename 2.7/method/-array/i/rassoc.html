instance method Array#rassoc

rassoc(obj) -> Array | nil[permalink][rdoc]

自身が配列の配列であると仮定して、要素の配列でインデックス 1 の要素が obj に等しいものを検索し見つかった最初の要素を返します。該当する要素がなければ nil を返します。

比較は == 演算子を使って行われます。

[PARAM] obj:
検索するオブジェクトを指定します。 
例

a = [[15,1], [25,2], [35,3]]
p a.rassoc(2)    # => [25, 2]
