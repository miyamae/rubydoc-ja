instance method CSV::Row#field?

field?(data) -> bool[permalink][rdoc]

自身に与えられた値が含まれている場合は真を返します。そうでない場合は偽を返します。

[PARAM] data:
この行に含まれているかどうか調べたい値を指定します。 
例

require "csv"

row = CSV::Row.new(["header1", "header2", "header3", "header4"], [1, 2, 3, 4])
row.field?(1) # => true
row.field?(5) # => false
