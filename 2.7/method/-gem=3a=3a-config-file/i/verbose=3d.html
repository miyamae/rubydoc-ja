instance method Gem::ConfigFile#verbose=

verbose=(verbose_level)[permalink][rdoc]

ログの出力レベルをセットします。

以下の出力レベルを設定することができます。

false

何も出力しません。

true

通常のログを出力します。

:loud

より多くのログを出力します。

[PARAM] verbose_level:
真偽値またはシンボルを指定します。
