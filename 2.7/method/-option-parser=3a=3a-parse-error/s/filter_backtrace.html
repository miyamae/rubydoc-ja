singleton method OptionParser::ParseError.filter_backtrace

filter_backtrace(array) -> [String][permalink][rdoc]

array で指定されたバックトレースから optparse ライブラリに関する行を除外します。

デバッグモード($DEBUGが真)の場合は何もしません。

[PARAM] array:
バックトレースを文字列の配列で指定します。
[RETURN]
array を返します。
