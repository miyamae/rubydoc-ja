instance method OpenSSL::PKCS7#error_string

error_string -> String | nil[permalink][rdoc]

検証エラーの理由を表す文字列を返します。

OpenSSL::PKCS7#verify で検証をした場合のみ更新されます。

OpenSSL::PKCS7#verify で検証をする前は nil を返します。

検証に成功した場合は nil を返します。
