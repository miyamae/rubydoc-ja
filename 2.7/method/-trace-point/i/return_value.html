instance method TracePoint#return_value

return_value -> object[permalink][rdoc]

メソッドやブロックの戻り値を返します。

[EXCEPTION] RuntimeError:
:return、:c_return、:b_return イベントのためのイベントフックの外側で実行した場合に発生します。 
例

def foo(ret)
  ret
end
trace = TracePoint.new(:return) do |tp|
  p tp.return_value # => 1
end
trace.enable
foo 1
