Gem::Command で使用する --local, --remote オプションのためのモジュールです。

accept_uri_http[permalink][rdoc]

HTTP の URI を扱えるようにするためのメソッドです。

add_bulk_threshold_option[permalink][rdoc]

add_local_remote_options[permalink][rdoc]

both? -> bool[permalink][rdoc]

ローカルとリモートの情報を両方とも取得する場合は真を返します。そうでない場合は偽を返します。

local? -> bool[permalink][rdoc]

ローカルの情報を取得する場合は真を返します。そうでない場合は偽を返します。

remote? -> bool[permalink][rdoc]

リモートの情報を取得する場合は真を返します。そうでない場合は偽を返します。
