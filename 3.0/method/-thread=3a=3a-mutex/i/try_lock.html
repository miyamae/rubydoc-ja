instance method Thread::Mutex#try_lock

try_lock -> bool[permalink][rdoc]

mutex をロックしようとして、ロックが成功した場合、真を返します。ロックできなかった場合にはブロックせず偽を返します。 

例

m = Mutex.new
m.try_lock # => true
m.try_lock # => false
