singleton method ENV.to_h

to_h -> Hash[permalink][rdoc]
to_h {|name, value| block } -> Hash

環境変数の名前をキーとし、対応する値をもつハッシュを返します。

ブロックを指定すると各ペアでブロックを呼び出し、その結果をペアとして使います。 

ブロック付きの例

ENV.to_h {|name, value| [name, value.size] }
