singleton method GC.enable

enable -> bool[permalink][rdoc]

ガーベージコレクトを許可します。

前回の禁止状態を返します(禁止されていたなら true, GC が有効であったなら、 false)。

[SEE_ALSO] GC.disable 

例

GC.disable   # => false
GC.enable    # => true
GC.enable    # => false
