instance method REXML::Namespace#has_name?

has_name?(other, ns = nil) -> bool[permalink][rdoc]

self が other と ns で指定した名前を持っているならば真を返します。

ns が nil でない場合は名前空間も比較されます。 other が ":" を含んでいる(つまり prefix を含んでいる)場合は REXML::Namespace#fully_expanded_name と other を比較します。どちらでもない場合は other と REXML::Namespace#name を直接比較します。

[PARAM] other:
比較する名前(文字列)
[PARAM] ns:
比較する名前空間(文字列)
