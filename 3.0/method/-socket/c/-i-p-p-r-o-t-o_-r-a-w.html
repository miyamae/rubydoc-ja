constant Socket::IPPROTO_RAW

IPPROTO_RAW -> Integer[permalink][rdoc]

Raw IP packets protocol。 BasicSocket#getsockopt, BasicSocket#setsockopt の level 引数に使用します。

また、Socket.open の protocol 引数に渡す利用法もあります。
