instance method IO#closed?

closed? -> bool[permalink][rdoc]

self が完全に(読み込み用と書き込み用の両方が)クローズされている場合に true を返します。そうでない場合は false を返します。 

例

IO.write("testfile", "test")
f = File.new("testfile")
f.close         # => nil
f.closed?       # => true
f = IO.popen("/bin/sh","r+")
f.close_write   # => nil
f.closed?       # => false
f.close_read    # => nil
f.closed?       # => true
