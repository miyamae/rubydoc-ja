instance method ObjectSpace::WeakMap#[]=

self[key] = value[permalink][rdoc]

引数 key から引数 value への参照を作成します。

[PARAM] key:
参照元のオブジェクトを指定します。
[PARAM] value:
参照先のオブジェクトを指定します。 
例

weak_map = ObjectSpace::WeakMap.new
key = "text"
weak_map[key] = "test" # => test
weak_map[key] # => test
