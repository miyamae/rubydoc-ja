instance method REXML::StreamListener#tag_start

tag_start(name, attrs) -> ()[permalink][rdoc]

開始タグをパースしたときに呼び出されるコールバックメソッドです。

[PARAM] name:
タグ名が文字列で渡されます
[PARAM] attrs:
タグの属性が"属性名" => "属性値"という Hash で渡されます

例

<tag attr1="value1" attr2="value2">

という開始タグに対し、 

name: "tag"
attrs: {"attr1" => "value1", "attr2" => "value2"}

という引数が渡されます。
