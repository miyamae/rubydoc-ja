instance method Numeric#zero?

zero? -> bool[permalink][rdoc]

自身がゼロの時、trueを返します。そうでない場合は false を返します。 

例

p 10.zero?              #=> false
p 0.zero?               #=> true
p 0.0.zero?             #=> true
