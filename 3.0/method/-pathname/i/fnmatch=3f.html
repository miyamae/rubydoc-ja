instance method Pathname#fnmatch?

fnmatch?(pattern, *args) -> bool[permalink][rdoc]

File.fnmatch?(pattern, self.to_s, *args) と同じです。

[PARAM] pattern:
パターンを文字列で指定します。ワイルドカードとして `*', `?', `[]' が使用できます。Dir.glob とは違って `{}' や `**/' は使用できません。
[PARAM] args:
File.fnmatch を参照してください。
