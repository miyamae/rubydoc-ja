instance method Pathname#split

split -> Array[permalink][rdoc]

File.split(self.to_s) と同じです。 

例

require "pathname"

pathname = Pathname("/path/to/sample")
pathname.split # => [#<Pathname:/path/to>, #<Pathname:sample>]
