instance method Kernel#xsystem

xsystem(command, opts = nil) -> ()[permalink][rdoc] [added by mkmf]

Kernel.#system と同じですが、コマンドの出力は(標準出力、標準エラー出力ともに)ログファイルに出力します。ログファイル名は mkmf.log です。
