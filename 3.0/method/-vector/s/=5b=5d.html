singleton method Vector.[]

self[*a] -> Vector[permalink][rdoc]

可変個引数を要素とするベクトルを生成します。

Vector[a1, a2, a3, ... ]としたとき、 引数a1, a2, a3, ... を要素とするベクトルを生成します。

[PARAM] a:
ベクトルの要素 
例

require 'matrix'
v1 = Vector[1, 3, 5, 7]
v2 = Vector[5.25, 10.5]
p v1   # => Vector[1, 3, 5, 7]
p v2   # => Vector[5.25, 10.5]
