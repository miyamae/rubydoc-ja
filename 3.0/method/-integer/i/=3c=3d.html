instance method Integer#<=

self <= other -> bool[permalink][rdoc]

比較演算子。数値として等しいまたは小さいか判定します。

[PARAM] other:
比較対象の数値
[RETURN]
self よりも other の方が大きい場合か、両者が等しい場合 true を返します。そうでなければ false を返します。 

1 <= 0    # => false
1 <= 1    # => true
1 <= 2    # => true
