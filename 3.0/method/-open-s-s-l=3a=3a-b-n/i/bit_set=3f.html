instance method OpenSSL::BN#bit_set?

bit_set?(n) -> bool[permalink][rdoc]

自身の n ビット目が立っているなら true を返します。 


require 'openssl'

OpenSSL::BN.new("129").bit_set?(0) # => true
OpenSSL::BN.new("129").bit_set?(1) # => false
[PARAM] n:
調べるビットの位置
