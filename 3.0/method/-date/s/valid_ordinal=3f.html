singleton method Date.valid_ordinal?

valid_ordinal?(year, yday, start = Date::GREGORIAN) -> bool[permalink][rdoc]

Date.jd、および Date.ordinal も参照してください。

年
[PARAM] yday:
年の日
[PARAM] start:
