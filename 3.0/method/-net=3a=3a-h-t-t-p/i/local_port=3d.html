instance method Net::HTTP#local_port=

local_port=(port)[permalink][rdoc]

接続に用いるローカルポートを設定します。

nil の場合システムが適当にローカルポートを決めます。

デフォルトは nil です。

[PARAM] port:
ローカルポート(数値、もしくはサービス名文字列) 
例

require 'net/http'

http = Net::HTTP.new("www.example.com")
http.local_host = "192.168.0.5"
http.local_port = "53043"

http.start do |h|
  p h.get("/").body
end
