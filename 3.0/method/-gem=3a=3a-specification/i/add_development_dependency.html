instance method Gem::Specification#add_development_dependency

add_development_dependency(gem, *requirements) -> [Gem::Dependency][permalink][rdoc]

この gem の DEVELOPMENT 依存性を追加します。この gem の開発時に必要となる gem を指定します。 


gem "rack", "~> 1.6", ">= 1.6.12"
[PARAM] gem:
依存する gem の名前か Gem::Dependency のインスタンスを指定します。
[PARAM] requirements:
バージョンの必要条件を 0 個以上指定します。デフォルトは ">= 0" です。

[SEE_ALSO] Gem::Specification#add_runtime_dependency, Gem::Dependency
