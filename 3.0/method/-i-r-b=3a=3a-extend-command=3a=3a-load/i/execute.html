instance method IRB::ExtendCommand::Load#execute

execute(file_name, priv = nil) -> nil[permalink][rdoc]

ファイル path を Ruby スクリプトとみなし、現在の irb インタプリタ上で実行します。

Kernel.#load と異なり、path の内容を irb で一行ずつタイプしたかのように、irb 上で一行ずつ評価されます。

[PARAM] file_name:
ファイル名を文字列で指定します。
[PARAM] priv:
真を指定した場合は実行は内部的に生成される無名モジュール上で行われ、グローバルな名前空間を汚染しません。
[EXCEPTION] LoadError:
読み込みに失敗した場合に発生します。
