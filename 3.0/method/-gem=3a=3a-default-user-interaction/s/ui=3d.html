singleton method Gem::DefaultUserInteraction.ui=

ui=(new_ui)[permalink][rdoc]

デフォルトの UI を新しくセットします。

デフォルトの UI を明確にセットしたことがなければ、シンプルなコンソールベースの Gem::UserInteraction を自動的に使用します。

[PARAM] new_ui:
新しい UI を指定します。
