instance method Gem::StreamUI#alert

alert(statement, question = nil) -> String | nil[permalink][rdoc]

INFO レベルのアラートを表示します。

[PARAM] statement:
表示する文字列を指定します。
[PARAM] question:
必要であれば質問を指定します。
[RETURN]
question を指定した場合は、それに対する回答を返します。 question を指定しない場合は nil を返します。
