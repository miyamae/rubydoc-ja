instance method CSV#lineno

lineno -> Integer[permalink][rdoc]

このファイルから読み込んだ最終行の行番号を返します。フィールドに含まれる改行はこの値には影響しません。 

例

require "csv"

csv = CSV.new("header1,header2\nrow1_1,row1_2")
csv.lineno # => 0
csv.readline
csv.lineno # => 1
