instance method Set#-

difference(enum) -> Set[permalink][rdoc]
self - enum -> Set

差集合、すなわち、元の集合の要素のうち引数 enum に含まれる要素を取り除いた新しい集合を作ります。

[PARAM] enum:
each メソッドが定義されたオブジェクトを指定します。
[EXCEPTION] ArgumentError:
引数 enum に each メソッドが定義されていない場合に発生します。 

require 'set'
p Set[10, 20, 30] - Set[10, 20, 40]
# => #<Set: {30}>
