singleton method Dir.empty?

empty?(path_name) -> bool[permalink][rdoc]

path_name で与えられたディレクトリが空の場合に真を返します。ディレクトリでない場合や空でない場合に偽を返します。 

例

Dir.empty?('.')      #=> false
Dir.empty?(IO::NULL) #=> false
require 'tmpdir'
Dir.mktmpdir { |dir| Dir.empty?(dir) } #=> true
[PARAM] path_name:
確認したいディレクトリ名。
