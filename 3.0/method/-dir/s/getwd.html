singleton method Dir.getwd

getwd -> String[permalink][rdoc]
pwd -> String

カレントディレクトリのフルパスを文字列で返します。

[EXCEPTION] Errno::EXXX:
カレントディレクトリの取得に失敗した場合に発生します(が、普通は失敗することはありません)。 
例

Dir.chdir("/tmp")   #=> 0
Dir.getwd           #=> "/tmp"
