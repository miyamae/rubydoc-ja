instance method ARGF.class#readbyte

readbyte -> Integer[permalink][rdoc]

自身から 1 バイトを読み込み整数として返します。既に EOF に達していれば EOFError が発生します。

[EXCEPTION] EOFError:
既に EOF に達している場合に発生します。 
$ echo "foo" > file
$ ruby argf.rb file

ARGF.readbyte  # => 102
ARGF.readbyte  # => 111
ARGF.readbyte  # => 111
ARGF.readbyte  # => 10
ARGF.readbyte  # => end of file reached (EOFError)
