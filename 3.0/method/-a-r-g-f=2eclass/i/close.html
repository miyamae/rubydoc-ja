instance method ARGF.class#close

close -> self[permalink][rdoc]

現在開いている処理対象のファイルをクローズします。開くファイルが残っている場合は次のファイルをオープンします。ただし、標準入力はクローズされません。 

$ echo "foo" > foo
$ echo "bar" > bar

$ ruby argf.rb foo bar

ARGF.filename  # => "foo"
ARGF.close
ARGF.filename  # => "bar"
ARGF.close
