module function Math.#lgamma

lgamma(x) -> [Float, Integer][permalink][rdoc]

log(|gamma(x)|) と、gamma(x) の符号を返します。

符号は +1 もしくは -1 で返されます。

[PARAM] x:
実数
[EXCEPTION] TypeError:
xに数値以外を指定した場合に発生します。
[EXCEPTION] Math::DomainError:
x に -∞ を渡した場合に発生します。
[EXCEPTION] RangeError:
xに実数以外の数値を指定した場合に発生します。 
例

Math.lgamma(0) # => [Infinity, 1]
