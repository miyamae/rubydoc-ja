module function Math.#atan2

atan2(y, x) -> Float[permalink][rdoc]

y / x の逆正接関数の値を返します。

[PARAM] y:
実数
[PARAM] x:
実数
[RETURN]
返される値の範囲は [-π, π] です。 
例

Math.atan2(1,0)   #=>  1.5707963267949
Math.atan2(-1,0)  #=> -1.5707963267949
[EXCEPTION] TypeError:
y, x に数値以外を指定した場合に発生します。
[EXCEPTION] RangeError:
y, x に実数以外の数値を指定した場合に発生します。
