instance method Complex#finite?

finite? -> bool[permalink][rdoc]

実部と虚部の両方が有限値の場合に true を、そうでない場合に false を返します。 

例

(1 + 1i).finite?                 # => true
(Float::INFINITY + 1i).finite?   # => false
