singleton method Net::POP3.APOP

APOP(is_apop) -> Class[permalink][rdoc]

bool が真なら Net::APOP クラス、偽なら Net::POP3 クラスを返します。

使用例: 

require 'net/pop'

pop = Net::POP3::APOP($isapop).new(addr, port)
pop.start(account, password) {
  ....
}
[PARAM] is_apop:
真の場合に Net::APOP を返します。
