instance method Gem::Requirement#prerelease?

prerelease? -> bool[permalink][rdoc]

何らかのバージョンがプレリリースのものであれば、true を返します。 


p Gem::Requirement.new("< 5.0").prerelease?   # => false
p Gem::Requirement.new("< 5.0a").prerelease?  # => true
