instance method Delegator#public_methods

public_methods(all = true) -> [Symbol][permalink][rdoc]

そのオブジェクトが理解できる public メソッド名の一覧を返します。

[PARAM] all:
偽を指定すると __getobj__ のスーパークラスで定義されたメソッドを除きます。
