singleton method WEBrick::HTTPResponse.new

new(config) -> WEBrick::HTTPResponse[permalink][rdoc]

HTTPResponse オブジェクトを生成して返します。

[PARAM] config:
設定を保存したハッシュを指定します。:HTTPVersion は必須です。

res = WEBrick::HTTPResponse.new( { :HTTPVersion => "1.1" } )
