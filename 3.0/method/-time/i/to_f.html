instance method Time#to_f

to_f -> Float[permalink][rdoc]

起算時からの経過秒数を浮動小数点数で返します。1 秒に満たない経過も表現されます。 


t = Time.local(2000,1,2,3,4,5,6)
p t                  # => 2000-01-02 03:04:05 +0900
p "%10.6f" % t.to_f  # => "946749845.000006"
p t.to_i             # => 946749845
