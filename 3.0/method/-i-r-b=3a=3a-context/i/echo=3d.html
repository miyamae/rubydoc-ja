instance method IRB::Context#echo=

echo=(val)[permalink][rdoc]

irb のプロンプトでの評価結果を表示するかどうかを設定します。

.irbrc ファイル中で IRB.conf[:ECHO] を設定する事でも同様の操作が行えます。

[PARAM] val:
true を指定した場合、irb のプロンプトでの評価結果を表示します。false を指定した場合は表示しません。
