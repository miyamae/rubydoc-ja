instance method Enumerable#drop

drop(n) -> Array[permalink][rdoc]

Enumerable オブジェクトの先頭の n 要素を捨てて、残りの要素を配列として返します。

[PARAM] n:
捨てる要素数。 
例

e = [1, 2, 3, 4, 5, 0].each
e.drop(3)             # => [4, 5, 0]
