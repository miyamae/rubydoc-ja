instance method CSV::Row#field_row?

field_row? -> bool[permalink][rdoc]

フィールド行であれば真を返します。そうでなければ偽を返します。 

例

require "csv"

header_row = CSV::Row.new(["header1", "header2"], [], true)
row = CSV::Row.new(["header1", "header2"], [1, 2])
header_row.field_row? # => false
row.field_row?        # => true
